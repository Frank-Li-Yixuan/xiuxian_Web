# VFX 可读性检查清单 v0.1

用于每次新增法术、Boss 技能、雷劫、灵宝效果时的人工和自动检查。

## A. 判定点

- [ ] 专注模式下 7px 判定点全程可见。
- [ ] 大范围法术不会覆盖玩家判定点 24px 内圈。
- [ ] 受击、复活、神魂状态下，玩家是否可被击中一眼可知。
- [ ] 双人同屏时 P1/P2 判定核心颜色不同。

## B. 敌弹

- [ ] 普通敌弹有白色核心和红/品红外圈。
- [ ] 敌弹绘制层高于大多数玩家法术。
- [ ] 火系法术不会把敌弹染成看不清的红块。
- [ ] 高速弹有更亮外圈或拖尾，但不扩大真实判定误导。

## C. 雷劫

- [ ] 雷劫预警圈至少提前 60 frame 出现。
- [ ] 真实命中半径和外部氛围圈有区别。
- [ ] 天雷不可清弹的视觉语言区别于普通敌弹。
- [ ] 雷劫落点高于掉落物、普通粒子、玩家火场。

## D. 法术

- [ ] 五雷正法连锁线不会遮挡连续敌弹。
- [ ] 八卦剑阵清弹反馈短促，不常驻大片光幕。
- [ ] 红莲业火内部透明，边界清楚。
- [ ] 袖里乾坤可吸收/不可吸收弹幕区别明确。

## E. 震屏与闪白

- [ ] 震屏只作用于 render camera。
- [ ] 敌弹仍活跃时全屏闪白 alpha <= 0.22。
- [ ] 多个震屏不会累加强度。
- [ ] 雷劫落雷前 0.5s 没有干扰预警读法的大震屏。

## F. 性能

- [ ] 粒子使用对象池。
- [ ] Medium 档同屏普通粒子不超过 800。
- [ ] 连锁爆炸超过阈值时合并。
- [ ] Low 模式有线框/低粒子替代方案。

## G. 工程边界

- [ ] `src/sim/**` 不创建粒子。
- [ ] `src/sim/**` 不播放音效。
- [ ] `src/sim/**` 不调用 `Math.random()`。
- [ ] 效果由 `EffectEvent` 驱动。
- [ ] Renderer 不修改 `SimState`。

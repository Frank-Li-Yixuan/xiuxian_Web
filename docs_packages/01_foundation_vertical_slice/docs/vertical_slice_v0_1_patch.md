# 《局内战斗垂直切片 v0.1》修正版要点

## 1. 废弃旧叫法

旧文档中“共享修为条触发时停顿悟”的描述不准确。

应替换为：

- **团队灵气经验条**触发时停顿悟。
- **个人修为条**触发境界进阶、瓶颈和雷劫。

## 2. 第一大阶段仍然成立

第一大阶段“青云山·妖潮初临”继续作为垂直切片：

- 1-1 山门试炼：基础怪、首次顿悟。
- 1-2 林间妖影：冲锋怪、精英狼妖。
- 1-3 雾中邪修：远程弹幕怪、第二次顿悟。
- 1-4 妖潮压境：高压综合测试、局内本命法宝奖励。
- 1-5 青云雷劫：Boss 青云劫灵。

## 3. v0.1 是否强制触发局内雷劫

为了让第一版就能验证修为系统，建议提供一个 Debug 开关：

```ts
debug.forceDynamicTribulationAtSegment = "stage_01_04";
debug.forcePlayerCultivationToBottleneck = true;
```

正式难度下，第一大阶段可以只让玩家看到修为增长，不强制练气破筑基。  
开发/测试难度下，可以在 1-4 或 Boss 前强行触发一次“练气破筑基·局内三九雷劫”。

## 4. UI 必须增加第二条进度

左侧道法区必须显示：

- 真元池。
- 法术槽。
- 团队灵气经验条：用于顿悟。
- 玩家个人修为条：用于境界。
- 当前境界层数：例如“练气三层”。

## 5. 掉落必须拆分

不要再把所有球都叫经验。

- 灵气球：加 Insight EXP。
- 真元球：回真元。
- 修为清气/天材地宝：加 Cultivation。
- 材料：局外炼丹炼器。
- 玉简：奖励池或功法碎片。

## 6. 奖励池中的“修为增长”只是选项之一

顿悟三选一中可以出现：

- “屏息凝神”：立刻获得少量修为。
- “吞纳清气”：接下来 30 秒周天吐纳速度提高。
- “天材地宝”：获得一个局内炼化材料。

但这些选择不能让“顿悟等级”变成“境界等级”。

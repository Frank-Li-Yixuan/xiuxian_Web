# 局内成长双轨制设计：灵气经验 ≠ 修为

版本：v0.1  
目标：避免把修仙境界简化成普通肉鸽等级，让局内同时拥有“法的成长”和“生命本质的成长”。

---

## 1. 两条进度条

### 1.1 灵气经验条 / Insight EXP

灵气经验是高频成长，用于肉鸽构筑。

来源：

- 普通怪掉落的小灵气球。
- 精英怪掉落的大灵气球。
- Boss 掉落的纯净灵气。
- 部分灵宝、功法、天赋提高灵气吸附或转化效率。

触发：

- 灵气经验条满后，进入“时停顿悟”。
- 双人各自三选一。
- 可以用公共气运点重 Roll。

奖励类型：

- 新法术。
- 法术升级。
- 局内临时功法。
- 天赋。
- 体质。
- 局内灵宝。
- 局内本命法宝。
- 丹药。
- 少量修为增长词条。

灵气经验的设计目标是让玩家频繁感到 Build 成型，不改变境界本质。

---

### 1.2 修为条 / Cultivation

修为是低频但关键的境界成长，用于生命本质质变。

来源：

- 局内自然吐纳：随时间缓慢增长。
- 根骨、主修功法、聚灵相关加成提高吐纳效率。
- 丹药：破境丹、小破境丹、培元类丹药。
- 天材地宝：Boss 或精英掉落。
- 特定顿悟词条：例如“屏息凝神”“吞纳清气”，但它们只是增加修为，不触发顿悟本身。

触发：

- 修为条满后，不出现普通三选一。
- 如果是小境界进阶，直接触发小周天反馈和小幅属性提升。
- 如果是大境界瓶颈，则进入雷劫逻辑。

突破结果：

- 精、气、神按境界倍率暴涨。
- 真元池上限提高。
- 神识带来的拾取、索敌、暴击、穿透能力提高。
- 后续境界可解锁额外灵宝槽、魂修能力、法相等。

---

## 2. 局内雷劫

局内雷劫是动态环境事件，不切地图。

触发条件：

- 玩家在 Run 内修为达到大境界瓶颈。
- 例如练气九层 → 筑基。

表现：

- 背景乌云压顶。
- 屏幕边缘变暗。
- BGM 变沉。
- 顶部出现“天象异变”警告。

机制：

- 当前怪物波次不停止。
- 雷劫作为额外机制叠加。
- 红色锁定光柱提前 1 秒出现。
- 天雷不可被常规法术摧毁。
- 八卦剑阵可削弱部分普通弹幕，但不能直接清天雷。

成功：

- 持续 20–30 秒后，最后一道雷罚化作造化清气。
- 全屏清场。
- 玩家回满生命和真元。
- 突破境界。
- 精、气、神发生面板质变。

失败：

- v0.1 建议不直接结束本局。
- 失败者获得“天伤”临时 Debuff。
- 修为回退到瓶颈前一定比例。
- 保留继续战斗机会，避免第一版挫败过强。

---

## 3. 局外雷劫关卡

局外雷劫是硬核期末考试。

触发条件：

- 洞府修为达到大境界瓶颈。
- 玩家点击“突破”。
- 消耗破境丹、护体法宝或其他准备资源。

关卡特征：

- 独立劫雷台。
- 没有普通怪。
- 没有肉鸽顿悟。
- 只能用局外积累的功法、法宝、丹药、基础面板硬抗。

结构：

1. 前段：纯走位雷柱，测试擦弹。
2. 中段：密集雷劫，测试护体法宝和丹药储备。
3. 末段：心魔劫，镜像自机使用玩家同类弹道。

失败：

- 突破失败。
- 已消耗的部分材料不返还。
- 境界不提升。
- 可以重新准备后再挑战。

成功：

- 局外境界永久提升。
- 解锁更高阶洞府建筑、功法、法宝升星上限。
- 下一局初始精气神显著提高。

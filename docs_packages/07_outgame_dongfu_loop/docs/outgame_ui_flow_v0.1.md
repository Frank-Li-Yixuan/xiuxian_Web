# 局外洞府 UI Flow v0.1

## 1. 归府结算页

目的：把死亡或通关转化为“满载而归”。

布局：

```text
[标题] 道友归府
[阶段] 青云山·妖潮初临 / 击败青云劫灵
[资源清单]
[稀有收获]
[首次通关奖励]
[按钮] 返回洞府
```

文案示例：

```text
肉身虽损，一缕真灵携宝归府。
本次收获已入库。
```

---

## 2. 洞府首页

卡片顺序：

```text
1. 境界卡
2. 聚灵阵收菜卡
3. 推荐闭关事项
4. 当前 Loadout
5. 模块入口：藏经阁 / 聚灵阵 / 炼丹房 / 炼器阁 / 劫雷台
```

推荐闭关事项最多显示 3 条。

---

## 3. 藏经阁页

Tab：

```text
[功] [法]
```

功页显示：

```text
功法名称
等级
修炼进度
是否主修
是否加入修炼队列
当前收益
下一层效果
```

法页显示：

```text
法术名称
熟练度等级
是否已装备到 J/K/L/I
永久升级材料
局内初始等级
```

## 7. 劫雷台页

未达瓶颈：

```text
修为未满，尚不可渡劫。
```

达瓶颈但材料不足：

```text
筑基丹不足 / 雷髓不足 / 灵石不足
```

材料齐：

```text
三九天劫将临
成功：突破筑基
失败：消耗筑基丹与部分材料，境界不变
[开始渡劫]
```

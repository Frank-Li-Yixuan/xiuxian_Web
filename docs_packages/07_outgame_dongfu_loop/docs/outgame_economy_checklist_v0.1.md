# 局外经济与闭环检查清单 v0.1

## A. 每局结束后是否有成长动作？

- [ ] 死在 1-1 后仍能炼制至少 1 个消耗丹，或推进 1 个功法进度。
- [ ] 死在 1-3 后至少能完成 1 个明确升级。
- [ ] 通关第一大阶段后至少能完成 2 个明确升级。
- [ ] Boss 首杀能解锁新目标，而不是只给普通材料。

## B. 资源是否职责清晰？

- [ ] 灵石是通用底层货币。
- [ ] 灵草主要进入炼丹。
- [ ] 矿石主要进入炼器。
- [ ] 妖丹横跨炼丹/炼器，但不能成为所有配方唯一瓶颈。
- [ ] 雷髓是第一大阶段 Boss 材料，用于突破和雷系成长。
- [ ] 残破玉简用于功与法，不用于法宝升星。

## C. 双轨成长是否被破坏？

- [ ] 聚灵阵产出修为，不产出灵气经验。
- [ ] 局外修为提升不触发顿悟三选一。
- [ ] 局内灵气经验不推进局外境界。
- [ ] 破境丹影响修为/突破，不影响 TeamInsightExp。

## D. 丹毒是否适度？

- [ ] 局内消耗丹不增加永久丹毒。
- [ ] 永久属性丹增加丹毒。
- [ ] 丹毒的前两个阶段只轻微影响体验。
- [ ] 清丹毒成本可接受。
- [ ] 丹毒不能让玩家卡死。

## E. 法宝升星是否能改变手感？

- [ ] 2 星必须有可感知变化。
- [ ] 3 星必须有小质变。
- [ ] 升星不只写“伤害 +5%”。
- [ ] 第一大阶段通关后可把至少一个法宝升到 2 星。

## F. Loadout 是否支持策略？

- [ ] 法术空槽合法。
- [ ] 丹药空槽合法但有提示。
- [ ] 至少有稳健、清怪、双人护法三类预设。
- [ ] 装备未解锁物品会失败。

## G. 联机资源是否不争抢？

- [ ] 在线双人基础资源各拿一份。
- [ ] 个人气运只影响个人稀有奖励。
- [ ] 队友拾取不降低自己的局外结算。
- [ ] 断线重连后 receiptId 防重复结算。

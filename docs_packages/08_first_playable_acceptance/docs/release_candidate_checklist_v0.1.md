# 《Release Candidate Checklist v0.1》

用于 First Playable RC 打包前的最终检查。

---

## 1. 版本范围

- [ ] v0.1 范围已冻结。
- [ ] Feature Scope Matrix 中所有 Must 已完成或有豁免。
- [ ] Optional/Deferred 没有偷偷进入阻塞路径。
- [ ] 未新增外部 CDN、外部字体、外部图片、外部音频。

---

## 2. 数据

- [ ] 所有 JSON 可解析。
- [ ] ContentRegistry 校验通过。
- [ ] 所有 ID 唯一。
- [ ] 所有引用有效。
- [ ] 默认 Profile 可加载。
- [ ] 默认 RunConfig 可生成。

---

## 3. Headless 与确定性

- [ ] FixedTickRunner 60 FPS。
- [ ] Gameplay 时间以 frame 表达。
- [ ] Same-seed replay 通过。
- [ ] Two-client harness 10 分钟无 hash mismatch。
- [ ] `src/sim/**` 无 `Math.random`。
- [ ] `src/sim/**` 无 DOM/Canvas/真实时间读取。

---

## 4. 局内战斗

- [ ] P1 可完整通关第一大阶段。
- [ ] P1/P2 本地双人可完整通关或死亡结算。
- [ ] 自动普攻正常。
- [ ] 4 个法术正常。
- [ ] 3 个丹药正常。
- [ ] 1-1 到 1-4 波次正常。
- [ ] Boss 青云劫灵三阶段正常。
- [ ] 精血渡魂正常。
- [ ] 掉落与拾取正常。

---

## 5. 双轨成长

- [ ] 灵气经验只触发顿悟。
- [ ] 修为只触发小层突破/瓶颈/雷劫。
- [ ] 顿悟界面可选择和重 Roll。
- [ ] 修为小层突破能提升面板。
- [ ] Debug 雷劫可触发并成功/失败结算。

---

## 6. UI/UX

- [ ] 玩家能区分生命、真元、灵气经验、修为。
- [ ] 法术槽 Ready/CD/不足状态清楚。
- [ ] 丹药炼化进度清楚。
- [ ] Boss UI 清楚。
- [ ] 雷劫 UI 清楚。
- [ ] 救援 UI 清楚。
- [ ] 顿悟界面双方明牌。

---

## 7. VFX/手感

- [ ] 判定点永远可见。
- [ ] 敌弹白芯可见。
- [ ] 雷劫预警可见。
- [ ] 震屏不影响 Simulation。
- [ ] 粒子不影响 hash。
- [ ] 1-4 高压段没有明显长时间卡顿。

---

## 8. 局外洞府

- [ ] 通关/死亡都能生成 Receipt。
- [ ] 基础资源 100% 带回。
- [ ] 资源钱包正确。
- [ ] 聚灵阵可收菜。
- [ ] 可炼丹。
- [ ] 可升星法宝。
- [ ] 可修功/研法。
- [ ] 可生成第二局 RunConfig。
- [ ] 第二局开局可见强化。

---

## 9. 试玩

- [ ] 至少 3 次单人完整流程。
- [ ] 至少 2 次本地双人完整流程。
- [ ] 至少 1 次失败结算流程。
- [ ] 至少 1 次 Debug 雷劫流程。
- [ ] 试玩报告已收集。
- [ ] 阻塞级 Bug 为 0。

---

## 10. RC 结论

```text
[ ] 通过，可以进入内部试玩。
[ ] 有条件通过，列出豁免项。
[ ] 不通过，回到对应 Gate 修复。
```

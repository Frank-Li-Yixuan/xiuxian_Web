# 《End-to-End Playtest Script v0.1》

本脚本用于验证 First Playable 不是若干功能碎片，而是一条完整可玩的体验链。

---

## 0. 测试准备

使用：

```text
Profile: default_profile.v0.1.json
RunConfig: debug_run_config.v0.1.json
Stage: stage_01_qingyun.v0.1.json
Difficulty: normal
Mode: single_player first, local_coop second
```

测试者记录：

```text
1. 是否知道当前目标？
2. 是否能看清危险？
3. 是否自然使用法术？
4. 是否自然提前吃丹药？
5. 是否理解灵气经验和修为的区别？
6. 是否愿意再来一局？
```

---

## 1. 第一轮：默认单人完整通关

### Step 1：进入洞府

预期：

- 洞府总览显示默认资源。
- 有“开始试炼/进入青云山”按钮。
- Loadout 默认使用“稳健开荒”。

默认 Loadout：

```text
主修：锐金诀 Lv.1
本命法宝：青霜飞剑 ★
灵宝：小周天剑阵 ★ + 八卦玉佩 ★
法术：五雷正法 / 八卦剑阵 / 空 / 空
丹药：回春丹 / 燃血丹 / 清心丹
```

### Step 2：进入 1-1 山门试炼

操作：

```text
WASD 移动
观察自动普攻
拾取灵气球
释放 J 五雷正法至少 1 次
```

预期：

- 自动飞剑清怪直观。
- 灵气经验条上涨。
- 真元不足/冷却状态可读。
- 1-1 结束触发第一次顿悟。

### Step 3：第一次顿悟

操作：

```text
选择一个法术/灵宝/功法奖励。
若出现空法术槽奖励，优先选择红莲业火或袖里乾坤。
```

预期：

- 顿悟界面暂停战斗。
- 选择后奖励生效。
- 战斗恢复没有卡死。

### Step 4：1-2 林间妖影

操作：

```text
观察狼妖冲锋预警。
用横向移动躲避。
尝试在精英前吃燃血丹。
```

预期：

- 冲锋怪制造横向压力。
- 燃血丹不是瞬间效果，有炼化过程。
- 精英死亡掉落大量灵气/材料。

### Step 5：1-3 雾中邪修

操作：

```text
躲避邪修三连弹。
使用八卦剑阵保命或开路。
```

预期：

- 弹幕开始有 STG 压力。
- 八卦剑阵能形成短暂安全窗口。
- 触发第二次顿悟。

### Step 6：1-4 妖潮压境

操作：

```text
在高压怪潮前吃回春丹或清心丹。
使用红莲业火/五雷正法清怪。
观察同屏敌人峰值和可读性。
```

预期：

- 同屏敌人多但不混乱。
- 法术连锁爆炸有爽感。
- 丹药提前使用有价值。
- 触发第三次顿悟或局内本命法宝奖励。

### Step 7：1-5 Boss 青云劫灵

操作：

```text
Boss Phase 1：读雷珠散射。
Boss Phase 2：躲落雷线和侧边狼妖。
Boss Phase 3：提前吃丹药，使用法术爆发。
```

预期：

- Boss 血条与阶段变化清晰。
- Boss 战 90–130 秒。
- Phase 3 有高压但不随机秒杀。
- Boss 死亡后进入结算。

### Step 8：结算

预期：

- 显示本局获得：灵石、灵草、矿石、妖丹、玉简、雷髓等。
- 明确提示资源 100% 带回洞府。
- 可返回洞府。

## 2. 第二轮：局外强化后再进局

### Step 9：回洞府收菜

操作：

```text
收取聚灵阵收益。
```

预期：

- 增加修为和灵石。
- 不增加灵气经验。

### Step 10：炼丹

操作：

```text
炼制回春丹 x1 或燃血丹 x1。
```

预期：

- 消耗灵草/灵石。
- 丹药进入库存。
- 可放入下局丹药槽。

### Step 11：炼器

操作：

```text
将青霜飞剑升至 2 星。
```

预期：

- 消耗矿石/妖丹。
- 青霜飞剑星级 UI 更新。
- 下局普攻效果增强。

### Step 12：藏经阁

操作：

```text
修炼锐金诀或研法五雷正法。
```

预期：

- 功法/法术永久成长。
- Loadout 中可看到变化。

### Step 13：生成第二局 Loadout

操作：

```text
选择强化后的青霜飞剑。
带上新炼丹药。
进入 stage_01_qingyun。
```

预期：

- 第二局开局火力、星级、丹药库存、法术等级至少有一项明显变化。
- 玩家能理解“上一局资源让这一局更强”。

---

## 3. 本地双人专测

### Step 14：双人进入 1-3 或 Boss

操作：

```text
P2 故意受击倒下。
P1 清出安全区域。
P1 靠近 P2 神魂，按住 H/Interact 渡魂。
```

预期：

- P2 进入神魂，不直接团灭。
- 聚灵阵范围和救援进度清晰。
- P1 生命持续扣除。
- P2 复活后 2 秒无敌。

---

## 4. Debug 雷劫专测

### Step 15：触发局内雷劫

操作：

```text
使用 debug profile 或 debug command：P1 修为到练气九层瓶颈。
吞服小破境丹。
触发局内三九雷劫。
```

预期：

- 灵气经验条不参与雷劫。
- 当前怪潮/Boss 不暂停。
- 雷劫红圈和内圈可读。
- 成功后全屏清场、回满、属性暴涨。

---

## 5. 试玩报告需要回答的问题

1. 第一局是否 5–7 分钟内完成？
2. Boss 战是否太短或太长？
3. 你是否自然使用了法术？为什么？
4. 你是否自然提前吃了丹药？为什么？
5. 你能否区分灵气经验和修为？
6. 你死后是否仍觉得有收获？
7. 回洞府后你是否知道该强化什么？
8. 第二局是否明显更强？
9. 哪个特效最爽？哪个特效挡视线？
10. 你是否愿意再打一局？

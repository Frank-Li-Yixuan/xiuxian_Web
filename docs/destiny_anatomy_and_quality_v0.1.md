# 天命结构与品质规范 v0.1

## 1. 天命数据结构

每个天命由以下部分组成：

```text
基础信息：id、name、quality、kind、allowedSlots
叙事信息：oneLine、description、tags
成立条件：eligibility
变异规则：mutation
效果投射：lifeSim、outerBattlefield、outgame、多模式 tags
显化事件：life manifestation hooks
```

## 2. 品质体系

| 品质 | 定位 | 设计要求 |
|---|---|---|
| 凡命 | 小幅倾向 | 通常作为副命或劫命，不作为主天命 |
| 良命 | 明确倾向 | 低代价，适合普通开局 |
| 奇命 | 机制倾向 | 有轻代价，有人生事件倾向 |
| 玄命 | 强机制 | 明显优缺点，需要命盘支撑 |
| 地命 | 改变路线 | 必须有支撑条件和代价 |
| 天命 | 极强天命 | 必须绑定劫数或承载问题 |
| 逆命 | 失败成长 | 必须有前期压制或缺陷 |
| 禁命 | 禁忌路线 | 必须有长期风险和阶段性反噬 |

## 3. 命力预算

沿用上版预算：

```text
凡命：正面 1–2
良命：正面 2–4，负面 0–1
奇命：正面 4–7，负面 1–2
玄命：正面 7–11，负面 2–5
地命：正面 11–16，负面 5–8
天命：正面 16–24，负面 8–14
逆命：正面 18–28，负面 10–18
禁命：正面 24–40，负面 15–30
```

## 4. 主天命与副天命

主天命负责“此生核心叙事”。副天命负责补充与偏折。

```text
主天命：天妒英才、苟道至尊、废灵逆命、丹道奇才、魔心暗种等
副天命：药性亲和、灵觉微明、过目不忘、手疾眼快等
劫命：命火早衰、先天体弱、宿业缠身、神魂不稳等
变异命格：妄承天机、天骄遭厄、隐杀之命、净莲藏影等
```

## 5. 变异命格的品质

变异命格不一定比原命格弱。

它的作用是“解释矛盾”，不是惩罚玩家。

例如：

```text
魔心暗种 + 清净琉璃心
→ 净莲藏影
```

这个变异反而是更稀有、更有故事的主命格。

## 6. 卡牌 UI 规范

每张卡显示：

```text
品质徽记
命格名称
一句话描述
正面摘要
负面摘要
人生影响标签
玩法影响标签
共鸣/冲突提示
锁定状态
```

Debug 模式额外显示：

```text
eligibility pass/fail
mutation source
support score
anti score
```

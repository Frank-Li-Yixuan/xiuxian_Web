# Origin as Storyline System v0.2

## 1. 身世是事件线，不是标签

v0.2 中，身世必须至少拥有：

```text
出生环境
家庭/社会关系
常见地点
资源限制
常见风险
主线事件线
隐藏命偏置
随身物偏置
玩法插曲倾向
18 岁结算方向
```

## 2. 身世事件线阶段

每个身世应有四段：

```text
earlyEcho      0–3 岁，家庭与体质印象
childhoodSeed  4–8 岁，环境塑形
youthConflict  9–13 岁，第一次冲突或线索
teenChoice     14–17 岁，主动选择路线
```

## 3. 示例：药铺学徒

```text
earlyEcho:
  药香、炉火、草药气味成为童年背景

childhoodSeed:
  分拣草药、误食灵草、识别药性

youthConflict:
  有病人求救，是否冒险试药

teenChoice:
  炉火失控，是否以自身气血稳炉
```

## 4. 示例：破落修士之后

```text
earlyEcho:
  家里有不能打开的旧木箱

childhoodSeed:
  木剑、旧经、祖坟青烟

youthConflict:
  游方道士识出旧物，是否隐瞒

teenChoice:
  旧敌线索出现，是否追查
```

## 5. 身世不能被隐藏命吞掉

即使玩家有“古雷真血”，如果他是药铺学徒，人生也应先表现为药铺线，再逐步渗入雷雨异常，而不是所有事件都变成雷雨。

# 《Codex 执行顺序 v0.1》

## 阶段划分

1. BAS-C001：资产授权策略、目录、manifest、校验脚本。
2. BAS-C002：下载/登记第一批 2D 战斗视觉资产。
3. BAS-C003：下载/登记第一批音效资产。
4. BAS-C004：建立 `/dev/2d-assets` 与 `/dev/audio-assets`。
5. BAS-C005：Canvas 资产 Registry 与播放基础设施。
6. BAS-C006：子弹、掉落、磁吸轨迹资源化。
7. BAS-C007：命中、受击、死亡、爆炸特效。
8. BAS-C008：法术、法宝、丹药、灵宝特效。
9. BAS-C009：人物/敌人动画接入。
10. BAS-C010：音效总线与事件映射。
11. BAS-C011：背景分层与关卡氛围。
12. BAS-C012：质量门槛、截图、性能验收。

## 必须小步执行

不要一次执行全部。每个阶段都必须：

```text
给出修改文件
给出资源来源
给出测试结果
给出截图/预览路径
说明是否改了 src/sim/**
```

## 推荐先执行

```text
BAS-C001
BAS-C002
BAS-C003
BAS-C004
```

这四步完成前，不允许把资源接入正式战斗。

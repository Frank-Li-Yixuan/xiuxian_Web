# 《九宫命盘、三才阴阳与属性因果 v0.1》

## 0. 文档定位

本文件是 SIM-REDESIGN 十二件套的第 2 份，用于重构角色创建阶段的九宫属性体系。

它解决当前的问题：

> OAG、DT、HFO 已经能生成属性、灵根、天命、身世，但它们仍偏“模块拼接”。  
> 例如低悟性、低灵感、低根骨的人也可能 Roll 到【天妒英才】，这会让玩家觉得系统不讲理。

本文件的目标是把创建角色改成：

```text
随机属性
  → 推导三才/阴阳/五行
  → 推导命盘因果标签
  → 过滤/变异天命
  → 影响灵根、身世、隐藏血脉、人生事件
```

最终玩家看到的不是“随机数值 + 随机词条”，而是：

> 这一世的命盘有内在因果。

---

## 1. 九宫属性正式定义

九宫命盘包含九项：

```text
精、气、神
根骨、悟性、灵感
气运、心性、寿元
```

其中“精气神”是当下生命结构，“根骨/悟性/灵感/气运/心性/寿元”是先天资质与命数。

### 1.1 九项解释

| 属性 | 含义 | 主要影响 |
|---|---|---|
| 精 | 肉身、血气、体魄、承载 | 健康、生命、炼体、受击容错 |
| 气 | 真元、周天、吐纳、炼化 | 法术、修为、闭关、丹药火候 |
| 神 | 神识、识海、心念、魂魄 | 感知、梦境、索敌、心魔 |
| 根骨 | 骨相、身体资质、法体承载 | 修炼底盘、炼体、重伤恢复 |
| 悟性 | 学法、明悟、顿悟能力 | 功法、法术、读书、卡牌/DBG |
| 灵感 | 异象感应、梦兆、天机 | 隐藏血脉、奇遇、系统预兆 |
| 气运 | 福缘、命数、偶然性 | 掉落、保命、贵人、稀有事件 |
| 心性 | 道心、定力、抗诱惑 | 心魔、雷劫稳定、重大选择 |
| 寿元 | 命火、生机、闭关成本 | 短命天命、借寿、长期修行 |

### 1.2 数值范围

推荐范围：

```text
1–120
```

评级：

| 数值 | 评级 |
|---:|---|
| 1–19 | 破败 |
| 20–39 | 下等 |
| 40–59 | 平庸 |
| 60–74 | 尚佳 |
| 75–89 | 上乘 |
| 90–100 | 天资 |
| 101–120 | 非凡 |

> 100 以上不是普通强，而是“命盘异常”。它应该触发额外事件权重，而不只是更高数值。

## 2. 三才结构

九宫分三才：

```text
天宫：悟性、灵感、气运
人宫：精、气、神
地宫：根骨、心性、寿元
```

### 2.1 天宫

代表：

```text
天授之才、机缘、天机感应、异象
```

用于支撑：

```text
天妒英才
雷鸣识海
太阴入梦
福星高照
系统共鸣
天书残页
```

### 2.2 人宫

代表：

```text
现实生命结构，当前精气神是否成体系
```

用于支撑：

```text
以战养战
命悬一线
丹药承受
魂修出窍
域外战场开局属性
```

### 2.3 地宫

代表：

```text
承载、道心、命火，决定能否托住天赋
```

用于支撑：

```text
苟道至尊
大器晚成
清净琉璃心
废灵逆命
短命/借寿/病弱类命格
```

## 3. 阴阳结构

九宫还有阴阳偏性。

| 偏阳 | 偏阴 |
|---|---|
| 精、根骨、悟性 | 神、灵感、心性 |
| 气为流转 | 寿元为藏命 |
| 气运为变数 | 视命盘而阴阳不定 |

### 3.1 阳盛

特征：

```text
精、根骨、悟性高
```

倾向：

```text
剑修、炼体、火法、战斗、强突破
```

风险：

```text
急进、火毒、天道注视、心性不足
```

### 3.2 阴盛

特征：

```text
神、灵感、心性高
```

倾向：

```text
魂修、太阴、梦境、清心、符箓、苟道
```

风险：

```text
神游、梦魇、心魔、肉身不足
```

### 3.3 阴阳失衡

不是纯坏，它会生成强玩法：

```text
阳盛阴弱 → 战修、雷火、杀伐、早夭风险
阴盛阳弱 → 魂修、梦境、心魔、病弱
阴阳冲突 → 水火既济/走火入魔二分
```

## 4. 五行映射

九宫属性不直接等于五行，但会产生五行气机倾向。

| 五行/异性 | 支撑属性 | 路线 |
|---|---|---|
| 金 | 根骨、神、悟性 | 剑修、炼器、破甲、决断 |
| 木 | 精、气运、寿元 | 药理、生机、恢复、灵植 |
| 水 | 气、神、心性、寿元 | 真元、清心、护盾、隐匿 |
| 火 | 气、悟性 | 爆发、炼丹、灼烧、慧光 |
| 土 | 精、根骨、心性 | 阵法、承载、防御、聚灵 |
| 雷 | 灵感、气运、气 | 雷法、天劫、域外、爆发 |
| 阴 | 神、灵感、心性 | 魂修、梦境、太阴、心魔 |

### 4.1 灵根不应完全随机

灵根要受九宫影响。例如：

```text
根骨高、神高、悟性高 → 金倾向提高。
气高、悟性高 → 火倾向提高。
神高、灵感高 → 阴/水倾向提高。
灵感高、气运高、气高 → 雷倾向提高。
精高、心性高、根骨高 → 土倾向提高。
寿元高、精高、气运高 → 木倾向提高。
```

## 5. 派生评分

为了工程实现，九宫会派生六个评分。

### 5.1 talentScore 天赋慧光

```text
悟性 0.5 + 灵感 0.3 + 神 0.2
```

用于：

```text
天妒英才
早慧
过目不忘
雷鸣识海
符箓/DBG 路线
```

### 5.2 vesselScore 承载之器

```text
根骨 0.35 + 精 0.3 + 气 0.2 + 寿元 0.15
```

用于：

```text
炼体
龙骨未醒
器灵之眷
承受天命/血脉
```

### 5.3 stabilityScore 命盘稳定

```text
心性 0.45 + 气 0.2 + 神 0.2 + 寿元 0.15
```

用于：

```text
清净琉璃心
苟道至尊
抗心魔
雷劫稳定
```

### 5.4 destinyPressureScore 天道压力

```text
悟性 0.25 + 灵感 0.25 + 气运 0.2 + 寿元反向 0.2 + 心性反向 0.1
```

用于：

```text
天妒英才
天道注视
雷劫易至
系统前兆
```

### 5.5 lateBloomScore 后发底蕴

```text
寿元 0.4 + 心性 0.3 + 根骨 0.2 + 悟性 0.1
```

用于：

```text
大器晚成
坐忘常定
苟道至尊
闭关路线
```

### 5.6 rebellionScore 逆命潜力

```text
根骨反向 0.25 + 悟性反向 0.15 + 心性 0.3 + 灵感 0.2 + 气运反向 0.1
```

用于：

```text
废灵逆命
百折不摧
失败成长
逆命点
```

## 6. 不合理组合处理

### 6.1 不直接禁止，优先变异

当系统抽到某个强天命但命盘不支撑时，不一定直接删除，而应尝试变异。

例如：

```text
低悟性 + 低灵感 + 天妒英才
```

不要保留【天妒英才】，也不要简单 reroll。  
变异为：

```text
【妄承天机】
你似乎被高位天机注视，但自身并无承载之才。
偶有神异，更多时候只是灾兆。
```

这样反而更有故事。

### 6.2 典型变异

| 原天命 | 不合理条件 | 变异 |
|---|---|---|
| 天妒英才 | talentScore 太低 | 妄承天机 |
| 废灵逆命 | 天灵根+高根骨+高悟性 | 天骄遭厄 |
| 苟道至尊 | 同时强战斗/杀伐 | 隐杀之命 |
| 劫雷亲和 | 心性/寿元太低 | 雷厄印 |
| 丹道奇才 | 心性太低 | 躁火炼药 |
| 器灵之眷 | 神/气太低 | 法宝排异 |
| 魔心暗种 + 清净琉璃心 | 强冲突 | 净莲藏影 |

## 7. 对人生模拟的影响

九宫属性会输出人生事件偏置标签。

示例：

```text
悟性高 → 读书、旧书、顿悟、论道事件提高。
寿元低 → 病弱、咳血、借寿、短命预兆提高。
灵感高 → 梦境、异象、隐藏命、系统杂音提高。
根骨低 + 心性高 → 吐纳失败、逆命、百折不摧提高。
气运高 → 贵人、拾得旧物、奇遇提高。
神高 + 心性低 → 梦魇、心魔、禁忌低语提高。
```

这会让人生模拟符合角色底子。

## 8. 对后续模式的影响

九宫命盘并不只影响人生模拟，还影响各种试炼模式。

| 属性 | 域外战场 | 虫族入侵 | 万族试炼塔 | 天地棋局 |
|---|---|---|---|---|
| 精 | HP、救援、丹药承受 | 肉身抗压 | 生命上限 | 阵基耐久 |
| 气 | 真元、法术频率 | 技能频率 | 每回合能量 | 阵力 |
| 神 | 索敌、暴击、拾取 | 自动索敌 | 抽牌/预见 | 先手/识破 |
| 根骨 | 修为转化、护体 | 成长速度 | 战后恢复 | 棋子成长 |
| 悟性 | 顿悟质量 | 进化选项 | 卡牌升级 | 推演质量 |
| 灵感 | 隐藏事件 | 稀有进化 | 稀有卡/事件 | 奇阵 |

| 心性 | 抗心魔 | 围困抗压 | 净化状态 | 阵法稳定 |
| 寿元 | 借寿禁术 | 长期续航 | 禁忌牌资源 | 长线推演 |

## 9. 设计结论

九宫命盘的核心不是“数值面板”，而是：

> 解释这个人为什么会有这种命。

生成器必须从：

```text
随机属性 + 随机天命
```

升级为：

```text
命盘结构 → 天命成立/变异 → 人生事件偏置 → 试炼模式投射
```

这样玩家 Roll 到的每一世才会有因果感。

# 《凡域边地与修仙世界观底座 v0.1》

版本：v0.1
定位：0–18 岁人生模拟、角色创建、隐藏身世、半年选择、玩法插曲、18 岁觉醒之前的世界观总底座。
核心目标：让所有人生事件、天命显化、隐藏血脉、随身物、早期玩法插曲都发生在一个自洽的修仙世界中，而不是现代随机事件或无根文本。

---

## 0. 总结一句话

本游戏的开局人生模拟发生在：

> **玄元界 · 大胤王朝西南边境 · 青岚边郡 · 青云山脉外缘。**

这里不是大宗门核心，也不是完全无灵之地，而是一个凡俗与修真交界的边地：

```text
凡人村镇、药铺、私塾、道观、猎户、祖坟、山神庙、县城、镇妖司、散修、旧宗门遗迹、后山灵隙、妖兽传闻、域外裂隙预兆，共同构成 0–18 岁人生模拟的舞台。
```

玩家不是一出生就在宗门里开挂，而是在凡域边地长大，通过命盘、灵根、天命、身世、随身物和选择，逐渐靠近“修仙世界的背面”。

---

# 1. 世界总结构

## 1.1 世界名：玄元界

玄元界是一个灵气有高低差、修真势力与凡俗王朝并存的修仙世界。

它不是纯粹“人人修仙”的世界。大多数人一生仍在凡俗秩序中生活，真正接触修行的机会稀少而危险。

世界可分为五层：

| 层级 | 名称 | 说明 | 0–18 岁出现方式 |
|---|---|---|---|
| 第一层 | 凡域 | 凡人村镇、县城、农田、私塾、药铺、猎户、道观 | 主舞台 |
| 第二层 | 灵隙 | 灵气渗漏点、后山洞穴、古井、祖坟、荒祠、旧阵 | 隐藏事件、玩法插曲 |
| 第三层 | 修真边缘 | 散修、游方道士、镇妖司、坊市、低阶宗门外门 | 少年/青春事件 |
| 第四层 | 宗门世家 | 青云宗、丹霞谷、玄雷观、万剑山等 | 远景、传闻、试探 |
| 第五层 | 域外诸天 | 域外战场、万族试炼、系统征召、天地棋局 | 梦境/预兆/成年后 |

第一版人生模拟主要发生在前三层，第四层只作为远景诱惑，第五层只作为远景伏笔和玩法解释来源。

# 2. 开局地域：青岚边郡

## 2.1 青岚边郡的定位

青岚边郡位于大胤王朝西南边陲，北靠青云山脉，南接凡俗县镇，西面有古战场遗址与山中灵隙。

它的特点是：

```text
灵气不算浓，但不绝。
妖祸不算大，但不断。
宗门不在眼前，但影响遥远存在。
凡俗秩序稳定，但边地灾祸频仍。
```

这使它非常适合作为人生模拟起点：既能解释普通童年，也能解释偶发修仙事件。

## 2.2 主要地点

| 地点 | 说明 | 可触发事件 |
|---|---|---|
| 青石村 | 边地村落，玩家常见出生地之一 | 山村孤儿、猎户之子、妖兽传闻、山贼烟尘 |
| 青石县 | 小县城，有私塾、药铺、县衙、镇妖司分署 | 私塾童生、药铺学徒、县城见闻 |
| 青云山外缘 | 山脉外层，灵气偶有渗漏 | 后山异象、妖兽、旧洞府 |
| 玄溪 | 山脚溪流，水气与阴气交汇 | 水灵根、太阴、梦境事件 |
| 旧山神庙 | 香火断续，偶有灵应 | 功德、祖灵、道观、心魔事件 |
| 荒祠 | 不知供奉何物的破败祠堂 | 魔印、阴梦、禁忌选择 |
| 祖坟坡 | 旧族坟地，阴气重 | 身世、祖灵、旧物、前世线索 |
| 药王巷 | 青石县药铺与草药交易地 | 药铺学徒、丹道奇才、药铺铜炉 |
| 白鹿观 | 小道观，非大宗门，只懂浅法与斋醮 | 道观杂役、符箓、问心、灵根试探 |
| 镇妖司分署 | 王朝基层镇妖机构 | 妖兽事件、山贼、低阶妖祸 |
| 散修集 | 偶发集会，不稳定 | 旧书残页、游方道士、假药、真机缘 |

# 3. 凡域生活：玩家出生的现实层

0–18 岁模拟的第一体验应当是“凡人生活中夹杂不可解释的修仙异象”。

## 3.1 凡人社会结构

| 身份 | 说明 | 事件倾向 |
|---|---|---|
| 农户 | 青岚边地常见家庭 | 劳作、家境、山贼、妖兽 |
| 猎户 | 靠山吃山，接触妖兽传闻 | 练武、追踪、伤病、野性 |
| 药铺 | 识草药、分药、见病痛 | 药理、丹道、火候、丹毒 |
| 私塾 | 识字、读经、接触旧书 | 悟性、学识、残卷 |
| 道观 | 香火、斋醮、浅符、观礼 | 符箓、问心、功德 |
| 破落修士后人 | 有旧物但无完整传承 | 随身物、旧敌、前世 |
| 流民 | 灾荒、兵劫、漂泊 | 生存、气运、心性、业力 |
| 守墓人 | 接触阴气、生死、祖灵 | 阴灵根、魂修、心魔 |

## 3.2 凡域事件的基调

凡域事件应当具备以下特征：

```text
生活具体、危险含蓄、修仙异象不直白。
```

例如：

```text
不是：你发现自己是古雷真血，雷帝传人。
而是：暴雨夜，你骨节发热，远处雷声像有人在敲鼓。
```

```text
不是：你获得仙帝传承。
而是：祖宅角落的木剑，在你病重那夜自己颤了一下。
```

这能保持悬念，也能让隐藏血脉系统成立。

# 4. 灵隙：凡人与修仙的交界

## 4.1 灵隙是什么

灵隙是灵气、残阵、旧因果、妖气或域外气息渗入凡域的地方。

它们不是稳定秘境，而是“异常点”。

常见灵隙：

```text
古井
后山石洞
荒祠
祖坟
旧道观地下
山腹裂缝
雷击木
枯泉
废弃药炉
断剑石
```

## 4.2 灵隙的玩法功能

灵隙用于解释：

```text
为什么童年能触发 STG 小战斗
为什么梦中能进 DBG 试炼
为什么后山会有妖兽
为什么随身物会异动
为什么隐藏血脉有进度
为什么系统前兆会提前出现
```

灵隙中的玩法插曲不一定是真实大战，可以是：

| 类型 | 包装 | 适用玩法 |
|---|---|---|
| 真实遭遇 | 后山妖影、山贼、妖兽 | STG / 割草 |
| 灵识幻象 | 梦中域外、雷雨幻境 | STG / DBG |
| 问心试炼 | 道观符阵、祖灵问心 | DBG |
| 阵法推演 | 旧阵残盘、玉佩异动 | 自走棋 / 阵局 |
| 守护事件 | 药田虫潮、村口防守 | 割草 |

# 5. 修真势力：远景但有影响

0–18 岁阶段不建议让玩家直接进入大宗门。大宗门太强，会削弱凡域成长和系统开辟洞府的意义。

但修真势力必须作为远景存在。

## 5.1 青云宗

青云山脉深处的正道宗门。外门弟子偶尔下山巡视，凡人知道其名但难以接近。

事件功能：

```text
游方弟子、山中试炼、青云令残片、灵根初测、域外战场传闻。
```

## 5.2 丹霞谷

以丹道、药田、火候闻名。与青岚边地药材交易有关。

事件功能：

```text
药铺学徒、丹炉梦、丹圣遗骨、火候试炼。
```

## 5.3 玄雷观

雷法法脉，行事更接近斋醮、符令、镇邪。对雷灵根、天劫、古雷真血有远景关联。

事件功能：

```text
雷雨、符箓、天道注视、劫雷亲和。
```

## 5.4 万剑山

剑修法脉，遥远而锋锐。其旧弟子曾在青云山外缘留下断剑遗迹。

事件功能：

```text
残破木剑、前世剑魄、剑鸣、飞剑启蒙。
```

## 5.5 镇妖司

大胤王朝处理妖祸的基层机构。不是宗门，更多是凡俗武力与低阶法器结合。

事件功能：

```text
山贼、妖兽、村镇灾劫、练武、功德/业力分化。
```

# 6. 妖、鬼、魔、域外的早期使用原则

## 6.1 妖

早期可以真实出现，但强度要低。

适合：

```text
山魈、狼妖、草木小妖、虫潮、妖兽踪迹。
```

不适合早期：

```text
大妖王、灭城妖潮、仙级妖魔。
```

## 6.2 鬼与阴物

更多用于梦境、祖坟、荒祠、阴灵根、魂修线。

早期不要做恐怖游戏，而是：

```text
阴冷、旧梦、魂火、墓地低语、祖灵模糊庇护。
```

## 6.3 魔

魔不是简单邪恶怪物，而是“捷径、执念、反噬”。

早期表现：

```text
黑影低语、禁书、血誓、心魔种、魔印微痕。
```

## 6.4 域外

域外先作为预兆：

```text
天外战鼓
梦中黑色战场
耳边系统杂音
星空裂隙
战场残影
```

不要在 5 岁就把玩家完整丢进域外战场。儿童阶段可以通过梦境/灵识预演，青春阶段才允许更明确的系统前兆。

# 7. 系统的世界观解释

你们的游戏有“系统”，但修仙世界观中不能让它显得像现代手机 App。

建议称呼为：

```text
天机印
诸天试炼印
系统核心，UI 层可以叫系统
```

世界观解释：

> 系统并非现代科技，而是域外诸天战场遗留下来的“天机试炼机制”。它会在某些命盘异常者身上留下印记，通过可视化的方式向宿主显化任务、试炼、资源和洞府权限。

在凡域阶段，它表现为：

```text
耳边杂音
梦中冷光
天外战鼓
无字残页自现文字
命盘裂光
随身物被扫描
```

18 岁是否一定觉醒可以待定。v0.1 可默认 18 岁成年节点触发，但世界观允许：

```text
提前半觉醒
延后觉醒
拒绝征召
梦中预演
```

# 8. 0–18 岁阶段的年龄世界规则

## 婴幼年 0–3 岁

事件范围：

```text
家庭、命火、异象、病弱、梦境、随身物微动。
```

不建议触发真实战斗。

## 童年 4–8 岁

事件范围：

```text
读书、玩耍、采药、后山、道观观礼、梦境、轻微灵隙。
```

可以触发短玩法，但多为梦境/躲避/轻试炼。

## 少年 9–13 岁

事件范围：

```text
灵根显现、第一次冲突、游方道士、山贼、妖兽、祖传旧物。
```

可以触发较明确的玩法插曲。

## 青春 14–17 岁

事件范围：

```text
流派成形、随身物觉醒、血脉预兆、系统前兆、危险选择、心魔与入道。
```

是玩法插曲密度最高的阶段。

## 成年节点 18 岁

默认用于：

```text
系统觉醒 / 入道节点 / 域外征召 / 洞府权限前置
```

但后续设计中允许变体。

# 9. 可支持的开局主线

第一版人生模拟至少支持 8 条主线倾向。

| 主线 | 常见触发 | 主题 |
|---|---|---|
| 寒门读书线 | 私塾童生、悟性高 | 学识、残卷、问心、法术理解 |
| 药铺丹道线 | 药铺学徒、木火土、丹道奇才 | 草药、火候、丹炉、丹圣预兆 |
| 猎户练武线 | 猎户之子、精/根骨高 | 练武、妖兽、山林、伤病 |
| 道观香火线 | 道观杂役、心性/神高 | 符箓、功德、斋醮、祖灵 |
| 破落遗脉线 | 破落修士之后、旧物 | 残剑、残页、旧敌、前世 |
| 山村灾劫线 | 山村孤儿、灾星/天妒/劫运 | 山贼、妖兽、救人与逃亡 |
| 阴梦魂修线 | 阴灵根、守墓、太阴残脉 | 梦境、魂火、墓地、心魔 |
| 系统前兆线 | 系统共鸣体、天外战鼓 | 杂音、域外梦、试炼预演 |

这些主线不是职业锁定，而是事件权重和叙事倾向。玩家可以通过半年选择改变主线走向。

# 10. 世界观对事件设计的硬约束

## 10.1 禁止现代化

禁止早期事件出现：

```text
手机
电脑
现代学校
现代医院
现代公司
网游梗
现代警察
现代高楼
```

除非是系统 UI 的 meta 表达，不进入世界文本。

## 10.2 禁止过早大宗门化

0–18 岁不能轻易让玩家直接加入大宗门成为内门弟子。可以有：

```text
试探
传闻
外门弟子路过
游方道士
残令
考核线索
```

## 10.3 隐藏真名不泄露

创建和人生模拟早期不直接写：

```text
古雷真血
丹圣遗骨
系统共鸣体
前世剑魄
```

应写：

```text
雷云深处的战鼓
陌生火候
耳边杂音
木剑轻鸣
```

## 10.4 战斗玩法插曲必须有世界解释

不能突然出现“开始打雷霆战机小游戏”。必须包装为：

```text
灵识御物
梦中域外
后山妖影
符阵问心
虫潮守田
天地阵盘推演
```

## 10.5 强事件要有代价或限制

儿童阶段可出现神异，但不应直接给完美传承。强事件应伴随：

```text
伤病
心结
业力
寿元损耗
隐藏进度
后续风险
```

# 12. v0.1 结论

《凡域边地与修仙世界观底座 v0.1》的核心不是写完整小说设定，而是建立一套可以支撑事件系统的“世界规则”。

第一版要保证：

```text
玩家出生是凡人，但世界确实有仙。
修仙不常见，但异常足够多。
战斗插曲不是小游戏乱入，而是灵隙、梦境、试阵和灾劫的表现。
系统不是现代 App，而是域外天机试炼机制。
每条开局路线都有现实土壤：药铺、私塾、道观、猎户、祖坟、山村灾劫。
```

这份底座定住后，后续九宫命盘、天命、人生主线、月度事件、半年选择和 DeepSeek 文案生成才不会乱。

# 人生模拟世界事件规则 v0.1

## 1. 事件必须有世界位置

每个事件都需要至少一个 `settingTag`：

```text
village
county
medicine_shop
private_school
daoist_temple
back_mountain
ancestral_grave
abandoned_shrine
old_well
forest
river
spirit_rift
outer_dream
```

事件不能凭空出现。即使是梦境，也要说明梦境来源：天命、随身物、隐藏血脉、系统前兆。

## 2. 事件必须有真实性等级

| 等级 | 说明 | 例子 |
|---|---|---|
| mundane | 凡俗真实事件 | 读书、劳作、家事 |
| anomalous | 现实中的异常 | 雷雨骨痛、玉佩发烫 |
| dream | 梦境/识海事件 | 梦中战场、太阴旧梦 |
| trial | 试炼/幻阵 | 道观符阵、祖灵问心 |
| combat | 真实危险遭遇 | 山贼、妖兽、虫潮 |
| system_omen | 系统前兆 | 耳边杂音、天外战鼓 |

## 3. 玩法插曲必须说明包装

| 玩法 | 包装类型 | 适合年龄 |
|---|---|---|
| STG | 灵识御物、梦中域外、后山妖影 | 7 岁后，少年/青春更合适 |
| 割草 | 守田、虫潮、村口防守 | 10 岁后更合适 |
| DBG | 论道、符阵、问心、梦境 | 8 岁后可用 |
| 自走棋 | 阵盘、祖传玉佩、地脉推演 | 10 岁后可用 |

## 4. 低龄事件限制

0–3 岁：不得真实战斗。
4–8 岁：允许梦境/轻量躲避/短试炼。
9–13 岁：允许小规模真实冲突。
14–17 岁：允许核心玩法插曲和高风险选择。

## 5. 隐藏信息规则

事件日志中禁止泄露隐藏真名：

```text
禁止：古雷真血 +8
允许：你感到雷声离自己更近了一些。
```

内部效果可以写入结构化字段：

```json
{ "type": "hiddenFateProgress", "target": "hidden_fate_ancient_thunder_blood", "delta": 8 }
```

可见文本只写氛围。

## 6. 事件结果要保留因果

每个强事件必须至少影响一个长期状态：

```text
属性
人生技能
隐藏进度
随身物亲和
功德
业力
伤病
心结
age18Hook
```

不要只写纯文案。

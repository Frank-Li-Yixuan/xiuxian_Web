# 文风与世界观约束 v0.1

## 基础语气

- 古雅，但不要文言到难读。
- 具体而含蓄，不直接解谜。
- 每条日志只描述一个明确事件。
- 使用凡域边地设定：青石村、白鹿观、药铺、祖坟坡、青云山外缘、镇妖司、游方道士等。

## 禁止词与禁区

禁止现代化：

```text
手机、电脑、互联网、公司、地铁、现代学校、医院、警察、游戏、系统面板、小游戏
```

禁止隐藏泄露：

```text
古雷真血、丹圣遗骨、系统共鸣体、前世剑魄、魔印微痕、天书残页、域外战场回响
```

可替代表达：

| 内部真名 | 可见表达 |
|---|---|
| 古雷真血 | 雷云深处的战鼓、骨节发热、雷声靠近 |
| 丹圣遗骨 | 陌生火候、梦中丹炉、药香中的旧记忆 |
| 系统共鸣体 | 耳边断续杂音、冷光、无字残页自现字迹 |
| 前世剑魄 | 木剑低鸣、竹影如剑、梦中剑光 |
| 魔印微痕 | 荒祠黑影、不能念出的字、梦中低语 |

## 长度控制

- 月度日志：一段，不超过 100 字。
- 半年选择引入：不超过 160 字。
- 选项：尽量不超过 18 字。
- 阶段总结：3–5 句。
- 人生小传：分段输出。

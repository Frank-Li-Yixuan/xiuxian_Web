# 《半年重大选择中的玩法插曲系统 v0.1》

## 0. 定位

本系统定义：在 0–18 岁人生模拟中，**每半年一次的重大选择**何时、如何触发短玩法插曲。

此前的人生模拟结构是：

```text
每月自动事件
  → 每半年重大选择
  → 属性 / 隐藏命 / 身世 / 随身物 / 功德业力变化
```

现在升级为：

```text
每月自动事件
  → 事件线推进
  → 每半年重大选择
  → 选择可能直接结算
  → 也可能进入短玩法插曲
  → 插曲结果回写人生模拟
```

关键目标：

> 让玩家在关键人生节点“亲自经历一次试炼”，而不是只看文字结算。

---

## 1. 为什么需要玩法插曲

如果人生模拟只有文本事件和数值变化，长时间会变成：

```text
读日志
点选项
看属性加减
```

玩家很快会觉得自己只是旁观人生。

玩法插曲的作用是：

```text
1. 把关键选择变成可操作验证。
2. 让不同玩法提前教学：STG、割草、DBG、阵法推演。
3. 让玩家感觉自己的命盘真的影响玩法。
4. 为后续多模式试炼做自然铺垫。
5. 让半年选择产生“这次可能会发生大事”的期待。
```

但插曲不能过多，否则人生模拟会碎。所以本系统同时定义 **触发频率、年龄限制、持续时间、失败惩罚和跳过机制**。

---

## 2. 插曲的世界观解释

人生模拟中的玩法插曲不能被描述成“进入小游戏”。

应包装为：

| 玩法 | 世界观包装 | 示例 |
|---|---|---|
| STG 雷霆战机 | 灵识御物、梦中域外、后山妖影、飞剑异动 | 雨夜后山、域外梦战、飞剑试炼 |
| 肉鸽割草 | 守药田、村口防守、虫潮入侵、妖兽潮 | 虫潮入药田、村口妖影 |
| DBG 卡牌 | 符阵问心、旧书试炼、梦中论道、心魔辩法 | 游方道士问心、禁书一页 |
| 天地棋局 | 道观阵盘、祖传玉佩、地脉棋局、阵法推演 | 白鹿观试阵、祖传玉佩发光 |
| 文本判定 | 普通人生抉择 | 私塾读书、闭门静坐 |
| 阶段节点 | 入道、离家、拜师、系统预兆 | 离家之念、十八岁前兆 |

## 3. 插曲分层

半年选择的选项分为四层。

### 3.1 文本选择

直接结算，不进入玩法。

适合：

```text
读书
照顾家人
普通练武
日常采药
静坐修心
```

### 3.2 判定选择

不进入玩法，但使用成功判定公式。

适合：

```text
尝试吐纳
读残卷
压制病痛
调和心魔
说服他人
```

### 3.3 玩法插曲

进入短玩法，结果回写人生。

适合：

```text
夜探后山
守护药田
梦中论道
道观试阵
村口防守
系统预演
```

### 3.4 转阶段候选

不是普通奖励，而可能改变人生阶段或身份。

适合：

```text
离家
拜师
血脉半觉醒
系统半觉醒
获得入道资格
洞府梦境
```

## 4. 年龄限制

### 0–3 岁

禁止真实玩法插曲。

允许：

```text
梦境片段
命火异象
被动文本事件
```

不允许：

```text
真实战斗
真实 DBG
真实自走棋
```

### 4–8 岁

允许极短、低风险、梦境/幻象包装的插曲。

推荐：

```text
60 秒以内 STG 躲避
3 回合 DBG 问心
短判定小游戏
```

失败后：

```text
伤病轻微
心结轻微
不死亡
```

### 9–13 岁

允许第一次真实小规模历练。

推荐：

```text
90–150 秒 STG
2–3 分钟割草
5 回合 DBG
3–5 回合阵法推演
```

失败可带来：

```text
伤病
心结
家境损失
但也可给逆命点 / 战斗经验
```

### 14–17 岁

允许核心玩法插曲。

推荐：

```text
2–4 分钟 STG
3–5 分钟割草
6–10 回合 DBG
5–8 回合天地棋局
```

可影响：

```text
人生主线
隐藏命进度
随身物亲和
18 岁 hook
是否提前入道 / 半觉醒
```

### 18 岁节点

不属于本系统的普通插曲，而进入“阶段转化 / 成年觉醒”系统。

---

## 5. 插曲频率控制

0–18 岁共 36 次半年选择。建议目标：

| 类型 | 数量目标 |
|---|---:|
| 文本 / 判定选择 | 22–26 |
| 普通短玩法插曲 | 6–8 |
| 核心主线玩法插曲 | 2–4 |
| 转阶段候选 | 1–3 |

总玩法插曲建议：

```text
8–12 次之间
```

第一次游玩时可以更少：

```text
6–9 次
```

避免人生模拟被频繁打断。

---

## 6. 插曲触发规则

插曲由以下因素共同决定：

```text
当前年龄阶段
最近 6 个月月度事件 hooks
活跃人生主线
事件线状态
当前天命
灵根
身世
隐藏预兆
随身物
当前伤病 / 心结 / 功德 / 业力
玩家过去选择风格
插曲频率预算
```

公式：

```text
interludeWeight =
  baseWeight
  + sourceHookWeight
  + storylineWeight
  + threadStateWeight
  + destinyWeight
  + rootWeight
  + originWeight
  + itemWeight
  + agePhaseWeight
  - frequencyPenalty
  - cooldownPenalty
```

## 8. 插曲 RunConfig

玩法插曲必须生成独立 `LifeInterludeRunConfig`。

核心字段：

```ts
interface LifeInterludeRunConfig {
  interludeId: string;
  mode: "stg" | "horde" | "deckbuilder" | "formation_auto";
  seed: string;
  ageMonth: number;
  sourceChoiceId: string;
  sourceThreadId?: string;
  difficultyTier: "safe" | "steady" | "risky" | "dangerous" | "forbidden" | "destiny";
  durationTargetSeconds?: number;
  turnLimit?: number;
  playerProjection: LifeInterludePlayerProjection;
  scenario: LifeInterludeScenario;
  rewards: LifeInterludeRewardTable;
  failurePolicy: LifeInterludeFailurePolicy;
}
```

插曲使用人生阶段投射，不使用完整成年人战斗 Profile。

例如 7 岁 STG 插曲：

```text
不是成年本命法宝战斗，而是“灵识幻象 / 躲避妖影”。
伤害低，失败只给轻微伤病。
```

## 9. 插曲结果回写

插曲结果必须回写人生模拟，而不是独立结算。

结果类型：

```text
success
partialSuccess
failure
greatSuccess
hiddenSuccess
abandon
```

回写内容：

```text
属性变化
人生技能变化
伤病
心结
功德 / 业力
隐藏命进度
随身物亲和
事件线进度
事件线 tension
主线分数
age18Hooks
未来事件锁定 / 解锁
```

示例：

```text
雨夜后山 STG 成功
→ 灵感 +4
→ 古雷预兆 +10
→ 后山妖影事件线 progress +20
→ age18Hook outer_battlefield_omen +1
```

失败：

```text
→ 伤病：风寒入骨
→ 心结：惧雷
→ 但废灵逆命获得逆命点
```

## 10. 跳过和自动结算

玩家可能不想频繁玩插曲。应允许：

```text
自动结算
跳过插曲
降低玩法频率
```

但跳过不能完全等于手动成功。

推荐规则：

```text
跳过 = 使用同样成功判定公式，但失去“大成功”和“隐藏成功”上限。
```

例如：

```text
手动玩法最高可 hiddenSuccess
自动结算最高只能 normalSuccess
```

这样玩家有动力亲自玩，但不被强迫。

---

## 11. 插曲失败策略

人生模拟里不能轻易 Game Over。

失败主要带来：

```text
伤病
心结
家境下降
随身物受损
业力增加
隐藏命负向进度
主线 tension 上升
```

特殊天命可把失败转收益：

```text
废灵逆命：失败获得逆命点
百折不摧：失败后心性 +，伤病减轻
魔心暗种：失败可能转成魔念收益
苟道至尊：如果选择避战，失败概率降低
```

## 12. v0.1 MVP 范围

必须做：

```text
插曲数据结构
触发规则
频率预算
RunConfig 工厂
结果回写
LifeSimulation / MajorChoice 集成
开发调试页
```

玩法实现可以是 MVP：

| 模式 | MVP |
|---|---|
| STG | 60–120 秒 Canvas 小战斗 |
| 割草 | 可先 placeholder / later |
| DBG | 可先文本 + 简易卡牌 placeholder |
| 天地棋局 | 可先文本 + 简易阵盘 placeholder |

v0.1 不要求四种玩法全部完整。

但数据契约必须能支持四种。

# 13. 验收标准

实现后必须满足：

```text
1. 0–3 岁不会触发真实玩法插曲。
2. 4–8 岁只触发低风险/梦境包装插曲。
3. 9–13 岁可触发小规模历练。
4. 14–17 岁可触发核心插曲。
5. 36 次半年选择中，插曲次数受预算控制。
6. 插曲触发受主线、天命、灵根、身世、随身物影响。
7. 插曲结果回写人生模拟。
8. 插曲不会直接修改成年 Profile 战斗数据，除非通过 18 岁 hook。
9. 跳过插曲可自动结算，但收益上限低于手动。
10. 同 seed 可复现。

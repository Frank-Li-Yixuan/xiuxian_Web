# 天命人生显化 Hooks v0.1

## 1. 设计目标

天命必须在人生模拟中显化。一个天命如果 18 年都没有事件呼应，就只是创建页 Buff。

每个核心天命至少要有：

```text
童年显化
少年考验
青春关键事件
成年结算投射
```

## 2. 显化强度

| 阶段 | 显化方式 |
|---|---|
| 0–3 岁 | 异象、亲缘反应、命火、病弱、梦境 |
| 4–8 岁 | 童年事件、轻量试炼、早慧、藏拙、旧物微动 |
| 9–13 岁 | 第一次考验、游方道士、真实冲突、问心 |
| 14–17 岁 | 流派成形、玩法插曲、血脉/系统前兆 |
| 18 岁 | 系统结算与模式投射 |

## 3. 示例

### 天妒英才

```text
幼年：无师识字。
童年：读旧经后高烧。
少年：看穿残缺口诀。
青春：咳血悟道。
18 岁：首次顿悟品质提升，但天道注视增加。
```

### 苟道至尊

```text
童年：躲过横祸。
少年：发现隐蔽石洞。
青春：闭门静坐积累潜修层数。
18 岁：存活奖励提高，主动施法削弱。
```

### 废灵逆命

```text
童年：吐纳失败却更耐痛。
少年：被断言无仙缘。
青春：失败后梦见锁链断裂。
18 岁：濒死触发逆命回响。
```

## 4. 与玩法插曲的连接

天命显化事件可以触发玩法插曲，但不要过度频繁。

```text
天妒英才 + 劫雷亲和
→ 雷雨后山 STG 插曲

丹道奇才
→ 守药田割草 / 炼丹火候小挑战

魔心暗种
→ 心魔 DBG 牌局

器灵之眷 + 天生剑骨
→ 飞剑灵识 STG 插曲
```

## 5. 事件文本规范

不直接说：

```text
你的天妒英才触发了。
```

而写：

```text
你只看了半页旧经，却像读过很多遍。夜里高烧时，窗外雷声一阵比一阵近。
```

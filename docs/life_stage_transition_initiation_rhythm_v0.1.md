# 人生阶段转化、入道节点与节奏控制 v0.1

## 0. 文档定位

本文件定义 0–18 岁人生模拟中的“阶段转化”系统。

它回答以下问题：

1. 是否必须 18 岁转阶段？
2. 什么时候算“入道”？
3. 玩家能否提前入道、延后入道、拒绝系统征召？
4. 半年重大选择和玩法插曲如何推动阶段变化？
5. 如何避免人生模拟被过多玩法插曲打断？
6. 如何把“凡人童年 → 半修行者 → 入道者 → 洞府之主”的过程做得合理、有节奏、有仪式感？

本系统承接：凡域边地世界观、九宫命盘系统、天命成立与变异系统、人生主线与事件线系统、半年重大选择中的玩法插曲系统。

---

## 1. 最高原则

### 1.1 年龄阶段 ≠ 修行身份阶段

人生模拟中有两条阶段线：

| 阶段线 | 是否固定 | 作用 |
|---|---:|---|
| 年龄阶段 Age Phase | 基本固定 | 决定事件池、文本口吻、可触发玩法强度 |
| 修行身份 Identity Stage | 动态变化 | 决定玩家是否接触修行、是否可触发玩法、是否可入道 |

例如：

```text
10 岁一定处于少年年龄阶段；
但他可能仍是凡人孩童，也可能已经是半修行者。
```

年龄只是时间。身份才是人生状态。

### 1.2 18 岁是默认成年大节点，不是硬编码唯一出口

默认情况下：

```text
18 岁 → 成年节点 → 系统觉醒 / 入道试炼 / 洞府开启前置
```

但设计上允许三种变化：

| 类型 | 说明 |
|---|---|
| 提前触发 | 隐藏命进度、系统前兆、关键玩法大成功等可提前触发半觉醒 |
| 正常触发 | 18 岁成年节点默认进行人生结算 |
| 延后触发 | 苟道、坐忘、隐修路线可延后进入正式试炼，v0.1 先作为预留 |

v0.1 工程实现可先采用：

```text
18 岁必定进入成年结算；
但成年结算根据人生状态选择不同“入道/系统试炼路径”。
```

### 1.3 入道是可解释的身份转化

“入道”不只是 stage = next。它应该由以下至少一种方式支撑：

```text
首次感应灵气
通过道观/散修/家传试炼
隐藏血脉半觉醒
随身物被激活
经历梦境或灵识玩法插曲
大灾大难中做出关键选择
系统前兆累计到阈值
18 岁成年节点强制结算
```

如果玩家只是每月随机 +2，到了 18 岁突然“系统觉醒”，会很硬。

### 1.4 玩法插曲可以推动阶段，但不能泛滥

玩法插曲是高亮节点，不是日常月报。推荐完整 0–18 岁：

```text
总半年选择：36 次
玩法插曲：8–12 次
关键转阶段插曲：2–4 次
正式成年试炼：1 次
```

## 2. 年龄阶段 Age Phase

### 2.1 婴幼年：0–3 岁

用途：命火、家庭、体质、异象、随身物微动。限制：不触发真实战斗、不触发域外、不触发割草，最多有梦境/异象文本。

### 2.2 童年：4–8 岁

用途：性格、初步学习、童年机缘、轻量玩法试探。允许短 STG 梦境、3–5 回合 DBG 问心、后山迷路、道观试香、孩童追逐，但必须包装为梦境、灵识幻觉、孩童试炼、轻度危险。

### 2.3 少年：9–13 岁

用途：灵根显现、首次冲突、身世线索、第一场小历练。允许小规模真实战斗、山贼烟尘、妖兽袭村、游方道士试阵、祖坟异象。

### 2.4 青春：14–17 岁

用途：流派初显、天命显化、系统前兆、关键玩法插曲。允许虫潮守药田、域外梦战、心魔试炼、阵法推演、飞剑异动、血脉半觉醒。

### 2.5 成年节点：18 岁

用途：人生模拟结算、系统觉醒 / 入道试炼 / 洞府权限试炼。v0.1 建议：18 岁触发成年结算界面，根据玩家当前身份与钩子，生成一条“成年试炼路径”。路径可为域外战场、宗门试炼、心魔问道、洞府开辟试炼等。默认 fallback 为域外战场第一战。

## 3. 修行身份阶段 Identity Stage

### 3.1 凡人孩童 mortal_child

初始身份。只经历凡域日常、家庭、体质、早期异象。不可主动修行，不可触发真实战斗插曲。

### 3.2 异象之子 omen_touched

角色已有异常，但未入道。梦境、异象、随身物、预兆事件权重上升。可触发低风险梦境玩法。

### 3.3 求道苗子 path_seed

角色开始主动接触修行。可触发读书、吐纳、道观、药理、练武相关重大选择。可积累 first_qi_sense / teacher_trial / artifact_resonance。

### 3.4 半修行者 half_initiated

已有实际修行经验，但还没有正式“入道”。可触发真实小规模玩法插曲。月度事件开始出现修行代价：伤病、走火、心魔、因果。

### 3.5 入道候选 initiation_candidate

角色已经接近真正入道。可触发入道节点。成年节点会依据此状态生成更强/更定制的转化路径。

### 3.6 入道者 initiated_cultivator

角色正式跨入修行门槛。可获得初始法门、初始灵识、修行身份标签。18 岁前若已入道，成年节点不再是“初次入道”，而是“试炼/洞府/系统权限”。

### 3.7 系统候选者 system_candidate

系统前兆已经显著。系统杂音、天外战鼓、命盘裂光、域外梦战权重提升。可与其他身份共存。

### 3.8 洞府之主 home_owner

成年试炼后状态。完成系统家园开启试炼或替代入道试炼并获得洞府权限后进入。

## 4. 入道节点 Initiation Node

v0.1 定义第一批节点：

| 节点 | 包装 | 主要来源 | 结果 |
|---|---|---|---|
| 初感灵气 first_qi_sense | 第一次明确感到天地灵气入体 | 吐纳、道观、静坐、山中灵隙 | path_seed → half_initiated |
| 游方试阵 teacher_trial | 游方道士/白鹿观布下试阵 | 道观、符箓、问心、DBG | half_initiated 或 initiation_candidate |
| 随身物共鸣 artifact_resonance | 残破木剑、祖传玉佩、无字残页异动 | 随身物、旧物、前世线 | 提升随身物亲和与身份 |
| 血脉半醒 bloodline_stirring | 隐藏血脉或前世残魂显著异动 | 隐藏命进度、高灵感 | initiation_candidate，风险增加 |
| 大灾择路 calamity_choice | 山贼、妖兽、虫潮、家变、村难 | 山村灾劫线、功德/业力 | 可能直接推进身份 |
| 系统预演 system_preview | 天外战鼓、命盘裂光、系统杂音、域外梦战 | 系统前兆线 | system_candidate |
| 问心入道 heart_trial | 梦中论道、心魔问答、禁书试心 | DBG/文本判定 | 心性、神、悟性变化 |

## 5. 阶段转化类型

阶段转化分 5 类：

| 类型 | 说明 | 示例 |
|---|---|---|
| 年龄转化 | 自动按年龄切换事件池 | 童年 → 少年 |
| 身份转化 | 由事件/选择/插曲触发 | 凡人孩童 → 异象之子 |
| 入道转化 | 完成入道节点 | 半修行者 → 入道者 |
| 系统转化 | 系统前兆/成年结算触发 | 系统候选者 → 域外征召 |
| 家园转化 | 成年试炼成功后 | 入道者 → 洞府之主 |

## 6. 节奏控制

### 6.1 插曲频率

完整 18 年建议：

```text
普通文本选择：22–26 次
玩法插曲候选：8–12 次
实际强制玩法：0–3 次
核心主线玩法：2–4 次
```

大多数插曲应允许：手动挑战、自动推演、暂避/放弃。

### 6.2 阶段转化冷却

```text
身份转化后 12 个月内不触发下一次身份转化，除非是强制灾劫或成年节点。
入道节点完成后 18 个月内不触发第二个入道节点。
系统预演类事件每 12 个月最多 1 次。
```

### 6.3 年龄强度限制

```text
0–3 岁：禁止玩法插曲。
4–8 岁：玩法不超过 60 秒/5 回合。
9–13 岁：玩法不超过 150 秒/6 回合。
14–17 岁：核心玩法可到 3–5 分钟/10 回合。
18 岁：成年试炼可独立设计。
```

## 7. 18 岁成年节点路径

v0.1 建议不再写死唯一“域外战场”，而是定义 5 个成年出口：

| 路径 | 适合条件 | 可用玩法 |
|---|---|---|
| 系统征召：域外战场 | 系统前兆线、雷灵根、天妒、劫雷亲和、高灵感 | STG |
| 入道试炼：凡域入道 | 道观香火线、破落遗脉线、入道候选 | DBG / STG / 阵法 |
| 灾劫清算：守护/逃亡 | 山村灾劫线 dominant、功德/业力高、亲缘深 | 割草 / STG |
| 血脉觉醒：内劫 | 隐藏血脉进度高、太阴/古雷/龙骨/魔印 | 心魔 DBG / 灵识 STG |
| 隐修延后：苟道/坐忘 | 苟道至尊、坐忘常定、高心性高寿元 | 延后模拟，v0.1 预留 |

## 8. 与玩法插曲系统集成

玩法插曲结果可能产生 transitionTokens：

| Token | 阶段影响 |
|---|---|
| stg_rain_mountain_success | omen_touched → path_seed |
| dbg_heart_trial_success | path_seed → half_initiated |
| horde_field_guard_success | 山村灾劫线推进，可能入道候选 |
| formation_jade_board_success | 阵法/道观线推进 |
| system_dream_survived | system_candidate 分数增加 |
| interlude_failure_fright | 心结增加，可能延后入道 |

## 9. UI 集成

人生模拟 UI 应显示两条阶段：

```text
年龄阶段：童年 / 少年 / 青春
修行身份：凡人孩童 / 异象之子 / 求道苗子 / 半修行者 / 入道候选 / 系统候选者
```

建议额外显示：

```text
入道机缘
系统前兆
主线张力
下一次阶段候选
```

当有阶段转化候选时，显示：

```text
天机微动：你似乎已经不再只是凡人。
[顺其自然] [压下异象] [主动探寻]
```

## 10. v0.1 MVP 范围

必须做：

```text
年龄阶段自动变化
身份阶段状态机
初感灵气 / 随身物共鸣 / 系统预演 / 灾劫择路 4 个入道节点
阶段转化 token
18 岁成年节点路径选择
默认域外战场 fallback
节奏预算：插曲与转化冷却
```

可以暂不做：

```text
完整延后觉醒路线
多宗门入门
复杂成年后继续凡域模拟
完整洞府开启多路径
```

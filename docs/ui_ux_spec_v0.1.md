# 人生模拟 UI / UX 规格 v0.1

## 1. 页面定位

LifeSimulationScreen 是“人生推演台”，不是普通日志页。

玩家应同时看到：

```text
时间
命盘变化
主线走向
近期事件
当前重大选择
隐藏预兆
随身物变化
```

## 2. 页面结构

### Header

显示：

```text
人生模拟
年龄：七岁六月
年龄阶段：童年
修行身份：异象之子 / 求道苗子等
速度控制
暂停
设置
```

### 左侧：命盘与天命显化

显示：

```text
九宫小面板
最近变化：精 +1、灵感 +2
当前显著天命：天妒英才 / 苟道至尊等
隐藏预兆：雷声、木剑、炉火等模糊词
```

### 中央：事件流与选择

常态显示月度日志流。

半年选择时切换为 MajorChoiceCard：

```text
标题
过去数月摘要
选项
风险标签
可能触发玩法提示
```

### 右侧：人生主线与随身物

显示：

```text
主线分数 TOP 3
事件线状态
随身物亲和
阶段转化候选
```

### 底部：时间轴

显示：

```text
0 岁 → 18 岁
阶段分段
重大选择节点
玩法插曲标记
当前月
```

## 3. 半年选择 UI

每个选项显示：

```text
风险字：稳/正/险/凶/禁/命
选择标题
一句描述
倾向：读书/练武/药理/隐藏预兆等
胜算等级：十拿九稳/吉凶参半/天机难测
玩法提示：可能进入灵识试炼
```

不显示精确数值概率。

## 4. 玩法插曲入口 UI

当选择携带 interludeCandidate：

```text
你选择了夜探后山。
这不是普通判定，将触发一段灵识试炼。

[手动挑战]
[自动推演]
[返回选择]
```

自动推演必须说明：

```text
自动推演最高只能获得“成功”，不会触发隐藏成功。
```

## 5. 结果反馈

选择或插曲结算后展示：

```text
发生了什么
可见变化
隐藏预兆变化，用模糊语
主线变化
随身物变化
新增心结/伤病/功德/业力
```

示例：

```text
你在雨中看见后山白雾里有影子掠过。归家后，你高烧两日，却总觉得雷声离自己更近了一些。

变化：灵感 +3，神 +1，新增心结：惧雷浅痕。
```

## 6. 日志回看

必须支持：

```text
全部日志
按阶段筛选
按主线筛选
按随身物筛选
只看重大选择
只看玩法插曲
```

首版至少支持“全部日志”和“重大事件”。

## 7. Reduced Motion

如果用户开启 reduced motion：

```text
日志淡入即可，不快速滚屏
禁用强镜头/强闪烁
插曲入口不做复杂转场
```

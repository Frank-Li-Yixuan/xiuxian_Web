# 《天命词条品质、互斥与重 Roll 系统 v0.1》

版本：v0.1
适用阶段：创建角色 / 推演天命 / 新存档开局
目标：让玩家在创建角色阶段获得“抽命、锁命、赌命、逆命”的 Roguelike 上头体验，并让这些结果真实影响 0–18 岁人生模拟、第一场域外战场和后续局外养成。

---

## 1. 系统定位

本系统解决四件事：

```text
1. 开局天命怎么随机。
2. 天命品质、效果、代价、互斥、共鸣如何定义。
3. 玩家如何重 Roll、锁定、推演、确认。
4. 天命如何影响人生模拟、局内战斗、多模式试炼和局外洞府。
```

创建角色不是“捏脸 + 加点”，而是：

> 玩家在推演一段修真人生的命盘。

玩家应该能在创建角色时产生这些心理：

```text
这个主命格太强，但缺陷也很要命。
这个废柴灵根好像能靠隐藏血脉逆袭。
这个组合看起来危险，但可能有隐藏共鸣。
我再 Roll 一次，万一出天命 + 合适劫命呢？
```

## 2. 术语定义

| 术语 | 说明 |
|---|---|
| 命盘 Draft | 创建角色时尚未确认的随机结果。 |
| 天命 Trait | 影响角色一生的机制词条。 |
| 主天命 Main Destiny | 最重要的核心词条，决定主玩法方向。 |
| 副天命 Secondary Destiny | 中等强度辅助词条，补充流派或放大特色。 |
| 劫命 / 缺陷 Calamity | 必带缺陷词条，制造代价、风险和故事张力。 |
| 隐命 / 隐藏血脉 Hidden Fate | 创建时只显示模糊预兆，人生模拟中逐渐揭示。 |
| 命力预算 Destiny Budget | 控制词条强度的内部预算。高收益必须有高代价。 |
| 互斥 Exclusive | 两个词条不能同时出现。 |
| 共鸣 Synergy | 两个词条同时出现时触发额外规则。 |
| 相冲 Conflict | 两个词条可共存，但会产生不稳定或副作用。 |
| 天机值 Fate Meter | 重 Roll 过程中用于避免一直出无聊低质量结果的保底/兴奋度机制。 |
| 天机锁 Lock | 玩家锁定某个结果，使其在重 Roll 中保持不变。 |

## 3. 开局槽位结构

每个角色开局命盘包含：

```text
主天命 × 1
副天命 × 2
劫命/缺陷 × 1
隐藏命/隐藏血脉 × 0–1
表面出身 × 1
随身物 × 1–2
灵根 × 1
先天资质 × 1 组
```

### 3.1 为什么必须有劫命

如果只给正面词条，玩家很快会追求纯数值最大化。劫命的作用是：

```text
1. 让每个开局有代价。
2. 让极端组合有故事。
3. 给人生模拟提供事件冲突。
4. 给后续心魔、寿元、雷劫、丹毒、业力等系统埋钩子。
```

劫命不是“坏东西”，而是修真故事中的“劫数”。大劫可能带来强因果，死劫甚至可能隐藏逆天机缘。

## 4. 天命品质体系

品质采用修仙语义，而不是普通 RPG 稀有度。

| 品质 | 英文 ID | 定位 | 视觉建议 |
|---|---|---|---|
| 凡命 | mortal | 小幅稳定影响 | 青灰 / 玉白 |
| 良命 | good | 明确方向，低代价 | 青玉 / 淡金 |
| 奇命 | rare | 机制倾向，有轻代价 | 翠青 / 银金 |
| 玄命 | mystic | 强机制，明显优缺点 | 紫青 / 金纹 |
| 地命 | earthly | 改变玩法，代价较大 | 深玉 / 赤金 |
| 天命 | heavenly | 极强命格，绑定劫数 | 金白 / 雷纹 |
| 逆命 | defiant | 失败越多越强、逆天改命 | 黑金 / 裂纹光 |
| 禁命 | forbidden | 禁忌强度，极高风险 | 黑紫 / 血金 |

### 4.1 品质概率

主天命不出凡命，保证开局有辨识度。

| 品质 | 主天命概率 | 副天命概率 |
|---|---:|---:|
| 凡命 | 0% | 25% |
| 良命 | 38% | 38% |
| 奇命 | 30% | 25% |
| 玄命 | 20% | 10% |
| 地命 | 8% | 2% |
| 天命 | 3% | 0% 基础，特殊保底/成就后开放 |
| 逆命 | 0.8% | 0% |
| 禁命 | 0.2% | 0% |

劫命独立按严重程度抽取：

| 劫命等级 | 概率 | 说明 |
|---|---:|---|
| 小劫 | 45% | 可控缺陷，轻微代价。 |
| 中劫 | 35% | 明显缺陷，影响路线。 |
| 大劫 | 17% | 强缺陷，但常带隐藏补偿。 |
| 死劫 | 3% | 极端挑战，需要明确提示。 |

## 5. 命力预算

高品质词条不能只有收益。每个天命都有正面预算和负面预算。

| 品质 | 正面预算 | 负面预算 | 设计规则 |
|---|---:|---:|---|
| 凡命 | 1–2 | 0 | 小加成。 |
| 良命 | 2–4 | 0–1 | 明确倾向。 |
| 奇命 | 4–7 | 1–2 | 有机制味。 |
| 玄命 | 7–11 | 2–5 | 强机制，有明显代价。 |
| 地命 | 11–16 | 5–8 | 改变玩法。 |
| 天命 | 16–24 | 8–14 | 极强，必须有劫数。 |
| 逆命 | 18–28 | 10–18 | 以失败、苦难、低开高走为核心。 |
| 禁命 | 24–40 | 15–30 | 禁忌玩法，必须高风险。 |

## 7. 重 Roll 系统

### 7.1 基础体验

创建角色阶段允许免费重 Roll。原因：开局抽命是爽点，不应卡死玩家。

但为了让重 Roll 更有策略，系统提供：

```text
1. 天机锁：锁定某个槽位，重 Roll 其余内容。
2. 天机推演：消耗有限次数，揭示隐藏预兆或提高下一次高品质概率。
3. 天机值：连续无高品质结果时积累，提升下次出现奇命以上概率。
4. 互斥/共鸣提示：帮助玩家判断组合价值。
```

### 7.2 默认资源

新角色创建时：

```text
免费重 Roll：无限
天机锁：2 次基础，可通过成就提升到 3
天机推演：1 次基础，可通过轮回印记提升
最多锁定槽位：2 个
```

可锁定槽位：

```text
灵根
主天命
副天命 1
副天命 2
劫命
出身
隐藏预兆（默认不可锁，需天机推演后可锁）
随身物
```

### 7.3 天机值

天机值用于避免玩家连续 Roll 到无趣命盘。

```text
初始：0
每次没有出现奇命及以上：+1
出现奇命：-2
出现玄命：重置为 0
出现地命及以上：重置为 0
天机值 ≥ 6：下一次主天命奇命以上权重提高
天机值 ≥ 12：下一次主天命至少奇命
```

注意：天机值不是“保底天命”，只是保证不太无聊。

## 8. 生成算法

### 8.1 输入

```ts
GenerateDestinyDraftInput {
  seed: string;
  slotId: string;
  lockedFields?: CharacterCreationLocks;
  previousDraft?: CharacterCreationDraft;
  metaProgress?: AccountMetaProgress;
}
```

### 8.2 输出

```ts
CharacterCreationDraft {
  spiritualRoot;
  coreStats;
  aptitude;
  destinies;
  background;
  hiddenFate;
  carriedItems;
  synergies;
  conflicts;
  warnings;
}
```

### 8.3 生成步骤

```text
1. 初始化 SeededRng。
2. 如果有锁定字段，先复制锁定结果。
3. 未锁定时生成命盘类型。
4. 生成出身。
5. 生成灵根。
6. 生成先天资质和出生三宝。
7. 生成主天命。
8. 生成劫命。
9. 生成副天命 1。
10. 生成副天命 2。
11. 检查互斥；若失败，按槽位重抽。
12. 计算共鸣与相冲。
13. 根据灵根、出身、命格抽隐藏命。
14. 根据出身与隐藏线索抽随身物。
15. 计算创建页摘要和预警。
```

### 8.4 槽位生成顺序为什么是主天命 → 劫命 → 副天命

先生成主天命是为了确定角色核心方向；接着生成劫命是为了尽早确定代价；副天命再围绕主方向、劫命和标签进行偏权重抽取。

这样更容易出现“有故事”的组合，而不是四个互不相关的词条。

## 9. 候选池过滤规则

抽取一个槽位时，按下列顺序过滤：

```text
1. slotType 是否匹配。
2. 是否满足 unlockCondition。
3. 是否与已锁定/已抽取词条互斥。
4. 当前槽位品质是否允许。
5. 是否超过高品质数量限制。
6. 是否满足成就/轮回解锁条件。
```

然后应用权重修正：

```text
最终权重 = 基础权重
  × 品质权重
  × 灵根标签权重
  × 出身标签权重
  × 命盘类型权重
  × 已有主命格标签共鸣权重
  × 天机值修正
  × 最近重复惩罚
```

## 10. 互斥、共鸣、相冲

### 10.1 互斥

互斥组合不能出现。

示例：

```text
天妒英才 × 大器晚成
苟道至尊 × 以战养战
清净琉璃心 × 魔心暗种（默认互斥，高级隐藏事件可破例）
天煞孤星 × 生死同契
丹道奇才 × 器道独尊
```

### 10.2 共鸣

共鸣组合允许，并显示“命格共鸣”。

示例：

```text
天妒英才 + 劫雷亲和
→ 雷劫更强，但雷劫成功奖励提高。

废灵逆命 + 百折不摧
→ 失败获得额外逆命点。

苟道至尊 + 坐忘常定
→ 潜修层数上限提高，但战斗收益进一步下降。

丹道奇才 + 饮鸩止渴
→ 丹药收益极高，但丹毒成为核心风险。
```

### 10.3 相冲

相冲组合允许，但增加不稳定性。

示例：

```text
魔心暗种 + 清净倾向类副命
→ 人生模拟中心魔事件与清心事件同时提高。

火灵根 + 寒毒类劫命
→ 丹药、修炼、体质事件出现冲突判定。
```

## 11. 对人生模拟的影响

天命必须影响人生模拟，否则只是创建页装饰。

### 11.1 影响类型

| 影响 | 说明 |
|---|---|
| 月度事件权重 | 某些事件更容易出现。 |
| 半年选择结果 | 某些选择收益提高或风险提高。 |
| 属性成长倍率 | 学习、练武、闭关、采药等结果改变。 |
| 隐藏进度 | 推动血脉、前世、系统共鸣揭示。 |
| 状态积累 | 心魔、功德、业力、伤病、心结。 |
| 特殊事件 | 命格专属事件。 |

### 11.2 示例映射

| 天命 | 人生模拟影响 |
|---|---|
| 天妒英才 | 学习/悟道收益大幅提高；病弱和天道注视事件提高。 |
| 苟道至尊 | 躲避、藏拙、闭关、不争选项收益提高。 |
| 以战养战 | 练武、冲突、救人、冒险收益提高；读书闭关降低。 |
| 丹道奇才 | 采药、药铺、炼丹、火候事件提高。 |
| 魔心暗种 | 梦魇、禁术、黑影、心魔选项提高。 |
| 清净琉璃心 | 心魔事件伤害降低，清修事件收益提高。 |

## 12. 对域外战场和多模式的影响

创建角色阶段的天命不只影响第一战，而是长期写入 Profile。

### 12.1 域外战场 STG

| 天命 | STG 表现 |
|---|---|
| 天妒英才 | 顿悟奖励质量提高，但高危事件/雷劫权重提高。 |

| 劫雷亲和 | 雷法增强，雷击事件更多。 |

| 丹道奇才 | 虫毒材料掉落和药效提高。 |
| 魔心暗种 | 虫潮密度提高，掉落也提高。 |

### 12.3 万族试炼塔 DBG

| 天命 | 卡牌表现 |
|---|---|
| 悟道类 | 初始卡牌升级机会增加。 |
| 借寿修行 | 可用寿元换真元/抽牌。 |
| 魔心暗种 | 心魔牌转化为收益牌。 |
| 清净琉璃心 | 状态牌净化能力增强。 |

### 12.4 天地棋局

| 天命 | 自走棋表现 |
|---|---|
| 阵道相关 | 阵位加成提高。 |
| 器灵之眷 | 器灵棋子更强。 |

## 14. 反作弊与可维护性

虽然本项目暂不商业化，也不强调服务器验证，但系统仍应保持可复现：

```text
同 seed + 同锁定 + 同 metaProgress = 同结果
重 Roll 记录保存在 draft history 中
确认此生后写入 confirmed character origin
人生模拟使用确认后的 seed 派生，不再受创建页重 Roll 影响
```

## 16. Codex 实现验收标准

必须满足：

```text
1. 使用 SeededRng，同 seed 可复现。
2. 主天命、副天命、劫命按不同权重抽取。
3. 锁定槽位后重 Roll 不改变锁定内容。
4. 不会抽出硬互斥组合。
5. 共鸣组合会被识别并显示。
6. 高品质词条必须带代价或警告。
7. 页面能显示正面/负面效果、标签、影响阶段。
8. 确认此生后写入 Profile origin。
9. 单元测试覆盖随机、互斥、共鸣、锁定、确认。
10. 不修改战斗模拟确定性边界。
```

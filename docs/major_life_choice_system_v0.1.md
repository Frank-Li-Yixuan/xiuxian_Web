# 《半年重大选择与成功判定系统 v0.1》

版本：v0.1
定位：0–18 岁人生模拟的玩家决策核心
目标：让每 6 个月一次的选择成为“人生路线塑造器”，而不是普通属性按钮。

---

## 1. 系统定位

月度事件系统每月自动推进，负责让角色的人生自然发生变化：

```text
出生异象
读书启蒙
雷雨预兆
药铺杂事
道观法事
山中争斗
梦境神识
身世线索
系统前兆
```

但如果 18 年完全自动，玩家只是旁观者。因此每 6 个月必须暂停一次，让玩家选择“如何回应这半年的命运”。

半年重大选择的作用是：

```text
1. 把最近 6 个月的月度事件汇聚成一个可决策节点。
2. 让玩家主动选择人生路线：读书、练武、采药、苟道、冒险、救人、禁忌、追查身世。
3. 让属性成长与命格、身世、隐藏血脉发生互动。
4. 埋下 18 岁系统觉醒和第一场域外战场的开局修正。
```

核心句：

> 月度事件告诉玩家“发生了什么”，半年选择让玩家决定“我成为什么样的人”。

---

## 2. 时间结构

```text
0–18 岁 = 216 个月
每月：自动事件
每 6 个月：重大选择
总计：36 次重大选择
18 岁：系统觉醒，不再触发普通半年选择
```

实际实现中，月度模拟可以快速播放；到每半年时暂停，让玩家选择。

```text
Month 0–5 自动推进
Month 6 重大选择
Month 7–11 自动推进
Month 12 重大选择
...
Month 216 系统觉醒
```

---

## 3. 重大选择的来源

半年选择不是完全随机，它由 5 层输入生成：

```text
1. 当前年龄阶段
2. 最近 6 个月月度事件 hook
3. 角色开局属性、灵根、天命、身世、隐藏血脉、随身物
4. 当前人生状态：伤病、心结、功德、业力、心魔、家境、亲缘、名声
5. 历史选择记录：是否长期苟修、是否好斗、是否救人、是否追查身世
```

例如：

```text
过去 6 个月出现“山贼烟尘”
  → 半年选择池加入“山贼逼近”事件。

过去 6 个月出现“梦中丹炉”并且是药铺学徒
  → 半年选择池加入“药炉异梦”事件。

拥有【苟道至尊】并连续 3 次选择避战
  → 半年选择池加入“藏拙潜修”特殊选项。

隐藏命为【古雷真血】且雷雨事件多次触发
  → 半年选择池加入“雨夜后山”隐藏选项。
```

## 4. 选择事件结构

每个半年节点生成一个 `MajorLifeChoiceEvent`。

它一般包含 3–4 个选项：

| 选项类型 | 风险 | 说明 |
|---|---|---|
| 稳健 | 低 | 小收益，几乎无负面 |
| 进取 | 中 | 中收益，需要判定 |
| 冒险 | 高 | 大收益，失败有代价 |
| 命格 / 隐藏 / 禁忌 | 特殊 | 条件触发，强收益或强代价 |

示例：

```text
【七岁·雨夜异象】
暴雨夜，你听见后山传来雷鸣般的低语。父母劝你不要出门。

选择：
1. 留在家中，闭门读书。        稳健
2. 偷偷前往后山查看。          冒险
3. 点燃香烛，向祖先祈告。      进取
4. 以龟息术藏住气息等待天明。  命格选项：苟道至尊
```

---

## 5. 风险等级

### 5.1 风险等级表

| 风险等级 | UI 文案 | 成功难度 | 失败代价 | 适用场景 |
|---|---|---:|---|---|
| safe | 稳 | 低 | 无或极小 | 读书、帮家、静养 |
| steady | 正 | 中低 | 轻微 | 练武、采药、拜访道观 |
| risky | 险 | 中高 | 伤病、心结、业力 | 探索后山、追敌、试药 |
| dangerous | 凶 | 高 | 重伤、心魔、寿元 | 禁地、妖兽、雷雨 |
| forbidden | 禁 | 极高 | 寿元、魔印、重业 | 禁术、借寿、血祭 |
| destiny | 命 | 条件特殊 | 视命格 | 命格、隐藏血脉专属 |

### 5.2 UI 显示

不要给玩家显示精确成功率到小数点。建议显示模糊等级：

| 成功率 | UI |
|---:|---|
| 85%+ | 十拿九稳 |
| 70–84% | 胜算颇高 |
| 55–69% | 可堪一试 |
| 40–54% | 吉凶参半 |
| 25–39% | 凶险 |
| <25% | 九死一生 |

某些隐藏选项显示：

```text
天机难测
```

## 6. 成功判定系统

### 6.1 基础公式

```text
score =
  d100
  + primaryStatScore
  + supportStatScore
  + skillScore
  + fortuneModifier
  + heartModifier
  + destinyModifier
  + originModifier
  + itemModifier
  - difficulty
  - woundPenalty
  - heartDemonPenalty
```

其中：

```text
d100 = 1–100
primaryStatScore = 主属性 × 0.35
supportStatScore = 辅助属性 × 0.20
skillScore = 相关人生技能 × 0.50
fortuneModifier = 气运 × 0.10
heartModifier = 心性 × 0.08
```

不同选项会声明自己的 primaryStats 和 supportStats。例如：

```text
读书：悟性、心性、学识
练武：根骨、精、武艺
采药：灵感、木灵根、药理
追查身世：灵感、见识、心性
山贼冲突：精、根骨、武艺、气运
压制心魔：神、心性、清净类天命
```

### 6.2 结果分层

| score | 结果 |
|---:|---|
| < 40 | 失败 |
| 40–64 | 有代价的成功 / 勉强成功 |
| 65–84 | 普通成功 |
| 85–99 | 大成功 |
| ≥100 | 大成功 + 额外奖励 |
| ≥100 且满足隐藏条件 | 隐藏成功 |

禁忌选项可能额外有：

```text
criticalFailureThreshold
```

例如禁术失败时会产生心魔、寿元损伤、隐藏魔印进度。

## 7. 选择结果类型

每个选择可以有 5 级 outcome：

```text
failure
mixed
success
great
hidden
```

### failure

```text
收益少或没有收益
伤病 / 心结 / 业力 / 心魔 / 随身物受损
但可触发废灵逆命、百折不摧等失败收益
```

### mixed

```text
获得部分收益，但伴随小代价
```

### success

```text
正常收益
```

### great

```text
额外属性、隐藏进度、随身物亲和、特殊 hook
```

### hidden

```text
触发隐藏血脉 / 前世 / 系统共鸣 / 命格特殊结果
```

## 8. 效果类型

本系统支持以下效果：

```text
modifyCoreStat         精 / 气 / 神
modifyAptitude         根骨 / 悟性 / 灵感 / 气运 / 心性 / 寿元
modifyLifeSkill        学识 / 武艺 / 药理 / 见识 / 名声 / 家境
modifyKarma            功德 / 业力
modifyHeartDemon       心魔
addWound               伤病
addHeartKnot           心结
addFlag                标记
modifyHiddenFate       隐藏命进度
modifyCarriedItem      随身物亲和
addMajorChoiceHook     后续选择 hook
addAge18Hook           18 岁系统觉醒 hook
addModeBiasTag         多模式试炼偏置
```

UI 中只显示可见效果，例如：

```text
悟性 +2
学识 +1
可能获得身世线索
有受伤风险
```

隐藏效果不显示真实名称。

## 9. 选择事件分类

半年重大选择分 12 类：

```text
study_path       读书学识
martial_path     练武护身
alchemy_path     采药药理
temple_path      道观法事
family_path      家庭亲缘
wilderness_path  山野探索
crisis_path      灾祸危机
dream_path       梦境神识
origin_path      身世追查
hidden_fate_path 隐藏血脉
destiny_path     天命触发
system_path      系统前兆
```

不同年龄阶段权重不同：

```text
0–3 岁：家庭、体质、异象
4–8 岁：读书、练武、采药、玩耍
9–13 岁：灵根、冲突、道观、身世
14–17 岁：流派成形、命格显化、系统前兆
```

## 10. 与命格的互动

### 苟道至尊

```text
避战、藏拙、闭关、静修选项收益提高。
连续避战获得潜修层数。
主动战斗、冒险、争斗选项会损失潜修层数。
```

### 天妒英才

```text
读书、悟道、修炼选项收益大幅提高。
但病弱、咳血、天道注视、雷劫 hook 提高。
```

### 废灵逆命

```text
失败不是纯负面。
重大选择失败时获得逆命点。
逆命点可在 18 岁结算中转化为根骨、心性或隐藏觉醒。
```

### 以战养战

```text
战斗、争斗、猎妖、护家收益提高。
避战、闭关、读书收益下降。
```

### 丹道奇才

```text
采药、药炉、试丹、火候选项收益提高。
试丹失败时丹毒风险降低。
```

### 魔心暗种

```text
禁忌、黑影、梦魇、借力选项收益提高。
心魔、业力、魔印进度增加。
```

## 11. 与隐藏血脉的互动

隐藏血脉在重大选择中只以“预兆”形式出现。

例如：

```text
古雷真血：
雷雨、后山、天道注视、域外战场梦境选项可能触发隐藏进度。

丹圣遗骨：
药炉、火候、试丹、药性冲突选项可能触发隐藏进度。

前世剑魄：
木剑、剑鸣、争斗、剑痕、破庙剑匣选项可能触发隐藏进度。

系统共鸣体：
耳边杂音、黑色战场、天外裂隙、时间错乱选项可能触发隐藏进度。
```

隐藏成功通常要求：

```text
score ≥ 100
对应隐藏进度 >= 某阈值
相关随身物亲和 >= 某阈值
或具备特定天命/灵根
```

---

## 12. 与随身物的互动

随身物不只是收藏品，会参与半年选择。

示例：

```text
残破木剑：
练武、剑鸣、山贼、前世剑魄事件中可触发。

祖传玉佩：
护身、祖灵、梦境、八卦玉佩线索中可触发。

药铺铜炉：
试药、火候、丹圣遗骨、药理选项中可触发。

无字残页：
读书、禁书、天书残页、系统共鸣选项中可触发。
```

随身物拥有：

```text
affinity 亲和度
damage 损伤
awakened 是否异动
conversionHook 18 岁转化
```

## 13. UI / UX 要求

半年选择 UI 必须清晰显示：

```text
事件标题
事件描述
发生年龄
选项列表
每个选项的风险等级
可见收益 / 可能代价
是否命格专属
是否隐藏条件触发
预计成功倾向
```

不要显示：

```text
隐藏血脉真名
精确随机公式
隐藏进度精确数字
```

示例卡：

```text
【十岁·山贼烟尘】
村外出现山贼踪迹，夜里有人看见火光。父母让你不要外出。

[稳] 留在家中，帮忙藏粮。
可见：心性 +，家境保全概率提高。

[险] 跟随猎户巡山。
可见：武艺 +，有受伤风险。

[凶] 夜里独自追踪火光。
可见：可能获得机缘，有重伤风险。

[命] 藏起气息，静待变局。
条件：苟道至尊
可见：潜修层数 +，错过争斗机缘。
```

## 14. 存档与恢复

每半年节点必须可保存。

```ts
interface PendingMajorChoiceState {
  eventId: string;
  generatedAtMonth: number;
  options: MajorChoiceOptionInstance[];
  selectedOptionId?: string;
  resolved?: boolean;
}
```

如果玩家退出，回来后应恢复：

```text
当前年龄
当前事件
当前选项
是否已选择
滚动日志
```

不要重新生成新选项，否则玩家会刷事件。

## 16. MVP 内容量

v0.1 推荐：

```text
重大选择事件：36 个
每个年龄阶段 8–10 个
每个事件 3–4 个选项
命格专属选项：至少 12 个
隐藏血脉选项：至少 12 个
随身物专属选项：至少 8 个
禁忌选项：至少 6 个
```

更小实现版本：

```text
事件 20 个
每阶段 5 个
命格专属 6 个
隐藏专属 6 个
随身物专属 4 个
```

## 17. 第一版验收标准

实现完成后，必须满足：

```text
1. 每 6 个月生成 1 次重大选择。
2. 选择事件受最近 6 个月 hook 影响。
3. 选择事件受天命、灵根、身世、隐藏命、随身物影响。
4. 每个选项有风险等级、成功判定、结果分层。
5. 结果可修改属性、伤病、心结、功德、业力、隐藏进度、随身物亲和。
6. 同 seed 结果可复现。
7. 未选择前存档恢复后事件不变。
8. UI 不泄露隐藏血脉真名。
9. 失败可触发废灵逆命等特殊收益。
10. 18 岁前可产生 age18 hooks。
```

# 隐藏血脉、身世盲盒与随身物叙事链 v0.2

## 1. 设计定位

v0.1 的 HFO 系统已经能生成：

- 表面身世
- 隐藏血脉 / 前世 / 封印 / 系统共鸣
- 隐藏预兆
- 随身物
- 18 岁转化钩子

但它还偏“创建页盲盒”。v0.2 的目标是把这三者变成贯穿人生模拟的叙事链：

```text
表面身世
  → 童年环境
  → 月度事件倾向
  → 半年选择分支
  → 玩法插曲钥匙
  → 隐藏命预兆
  → 随身物亲和
  → 18 岁揭示 / 半揭示 / 误导 / 封印
  → 域外战场 / 洞府初始方向
```

玩家应该感到：

> 我不是抽到了“古雷真血”这个标签，而是在多年雷雨、骨痛、后山异象、天外战鼓之后，逐渐意识到自己与雷有关。

## 2. 三条叙事链

### 2.1 身世链 Origin Chain

身世不是“山村孤儿 +2 气运”这种出生加成，而是决定玩家前 18 年最常接触什么世界：

| 身世 | 常见地点 | 事件倾向 | 隐藏命偏置 | 随身物偏置 |
|---|---|---|---|---|
| 山村孤儿 | 青石村、后山、山神庙 | 灾劫、雷雨、村难 | 古雷真血、系统前兆 | 残旧家书、童年石符 |
| 药铺学徒 | 药王巷、药铺、药田 | 草药、火候、救治 | 丹圣遗骨 | 药铺铜炉、雨渍旧经 |
| 道观杂役 | 白鹿观、香火殿、符阵 | 斋醮、符箓、问心 | 功德种子、系统共鸣 | 破旧符纸、旧香炉 |
| 猎户之子 | 青云山外缘、林地、溪谷 | 练武、狩猎、受伤 | 龙骨未醒、战场回响 | 锈弓残弦、残破木剑 |
| 破落修士之后 | 祖宅、旧书箱、祖坟坡 | 旧物、残卷、木剑 | 前世剑魄、天书残页 | 残破木剑、无字残页 |
| 守墓人之子 | 祖坟坡、荒祠、玄溪 | 阴梦、魂火、守夜 | 太阴残脉、魔印微痕 | 黑骨短笛、旧香炉 |

### 2.2 隐藏命链 Hidden Fate Chain

隐藏命具有多阶段显化：

```text
seeded       种子：创建时生成，不直接显示真名
omen         预兆：月度事件出现模糊异常
misleading   误导：出现可解释为其他隐藏命的假线索
stirring     异动：半年选择出现“命”选项
halfReveal   半揭示：玩家知道方向，但不知完整真名
revealed     揭示：真名写入 Profile
unstable     不稳定觉醒：强效果 + 风险
sealed       封印：未揭示，但保留线索
```

### 2.3 随身物链 Carried Item Chain

随身物有生命周期：

```text
obtained       取得
noticed        注意到
resonating     与事件/隐藏命共鸣
tested         通过选择或插曲验证
damaged        受损或代价
deepened       亲和加深
converted      18 岁被系统解析
inherited      转化为洞府线索/法宝/功法残篇
```

随身物应当陪玩家走过人生，而不是 18 岁突然变道具。

## 3. 揭示原则

### 3.1 不早泄露 trueName

创建页、早期月度事件、半年前期选择不得显示：

```text
古雷真血
太阴残脉
丹圣遗骨
前世剑魄
系统共鸣体
```

应显示为：

```text
雷云深处的战鼓
月下冰凉的影子
陌生火候
木剑里的低鸣
耳边断续杂音
```

### 3.2 允许误导

隐藏盲盒要有“猜测空间”。例如：

```text
雷雨骨痛
```

可能指向：

- 古雷真血
- 劫雷亲和
- 系统前兆
- 天道注视
- 妄承天机

系统内部知道真相，但 UI 只显示“雷雨相关预兆”。

### 3.3 半揭示比直接揭示更好

例如：

```text
你的血脉似与雷有关。
```

比直接写“古雷真血”更好。

18 岁前多数隐藏命应停留在：

```text
预兆 / 异动 / 半揭示
```

只有少数高进度才直接揭示。

## 4. 与玩法插曲的关系

随身物和隐藏命应当触发玩法插曲候选。

示例：

```text
残破木剑 + 木剑轻鸣事件线
→ 半年选择：[险] 夜里试剑
→ 触发 STG：竹林剑影

药铺铜炉 + 药炉火候事件线
→ 半年选择：[险] 守炉加火
→ 触发割草或判定：守护药田 / 火候试炼

祖传玉佩 + 祖玉阵局
→ 半年选择：[命] 按玉佩纹路摆阵
→ 触发天地棋局插曲
```

## 5. 与阶段转化的关系

隐藏命和随身物可以提供阶段转化 token：

```text
hidden_fate_stirred
bloodline_half_awakened
item_resonance_deepened
system_noise_answered
ancestral_object_recognized
```

这些 token 不直接改阶段，而是交给阶段转化系统判断。

## 6. 与 18 岁觉醒的关系

18 岁时，系统解析：

```text
隐藏命进度
随身物亲和
身世主线进度
玩法插曲胜负
半年选择关键结果
```

然后给出：

```text
revealedHiddenFates
sealedHiddenFates
convertedCarriedItems
outerBattlefieldModifiers
dongfuHooks
longTermTags
```

## 7. v0.2 成功标准

玩家应该在模拟过程中不断形成猜测：

```text
我好像和雷有关。
这把木剑肯定不普通。
这个药炉可能是丹道线索。
我耳边的杂音是不是系统？
这个黑骨短笛有点危险。
```

这比创建页直接列出“隐藏血脉：古雷真血”强得多。

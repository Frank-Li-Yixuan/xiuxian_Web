# 首版范围与裁剪 v0.1

## 必须进入首版

```text
LifeSimulationScreen
0–18 岁时间推进
月度事件自动播放
半年重大选择停顿
阶段总结
主线分数展示
随身物亲和展示
隐藏预兆不泄露
至少一个 STG 插曲入口和结果回写
插曲自动推演
模拟存档/恢复
216 月结束进入成年节点 pending
```

## 强烈建议进入首版

```text
雨夜后山 STG 插曲
天外战鼓 STG 插曲，至少 planned/auto
主线调试页 /dev/life-playable
人生日志回看
快速推演
阶段总结文本
```

## 不进入首版

```text
完整 DBG 插曲实现
完整割草插曲实现
完整天地棋局插曲实现
真实 DeepSeek API 必接
复杂 NPC 关系网
完整宗门拜师路线
完整成年后多分支
完整 18 岁域外战场重构
```

## MVP 内容下限

```text
月度事件池：可使用 v0.2 的 86 条
半年选择：可使用 v0.2 的 24 条
主线：至少 8 条评分可见
事件线：至少能推进 progress/tension/clarity
插曲：至少 1 条手动，2–4 条自动推演
阶段总结：4 个年龄阶段
```

## 切勿做的事

```text
不要用 LLM 决定事件结果。
不要让玩家每月都手点。
不要一进入人生模拟就触发战斗。
不要在早期泄露隐藏命真名。
不要把所有玩法插曲都做成强制。
不要在模拟中硬编码 18 岁必定域外战场。
```

# 域外战场第一战设计 v0.1

## 1. 定位

域外战场第一战是 18 岁系统觉醒后的第一场实战。

它不是完整常规模式，而是：

```text
新手教程
人生结果验证
天命效果展示
系统洞府开启条件
```

---

## 2. 叙事包装

```text
系统征召开始。
宿主将被投放至域外战场边缘。
目标：存活并击破战场巡游者。
奖励：系统家园权限。
```

场景关键词：

```text
虚空裂隙
漂浮战场碎片
天外残兵
妖魔残影
系统光幕
远处流星与残破战旗
```

---

## 3. 战斗时长

推荐：3–5 分钟。

| 阶段 | 时间 | 内容 |
|---|---:|---|
| 系统校准 | 0:00–0:30 | 移动、自动普攻、少量残兵 |
| 初次杀敌 | 0:30–1:20 | 小怪潮，掉落灵气/真元 |
| 第一次顿悟 | 1:20–1:40 | 强制顿悟三选一 |
| 命格显化 | 1:40–2:40 | 根据天命/灵根触发特殊事件 |
| 巡游者出现 | 2:40–3:50 | 小 Boss |
| 系统结算 | 3:50–4:20 | 敌人清场、家园权限开启 |

---

## 4. 教学点

必须自然教学：

```text
WASD 移动
自动普攻
拾取灵气
真元与法术
丹药消化
第一次顿悟
随身物/天命触发
Boss 警告
战斗结算
```

不要弹太多教程窗。优先使用系统提示短句。

---

## 5. 第一战初始配置

来自 Age18AwakeningResolution：

```text
初始本命法宝：由随身物 / 灵根 / 默认保底决定
初始法术：默认 1 个，受悟性 / 灵根 / 天命影响
初始丹药：0–2 个，受身世 / 随身物影响
初始符箓：0–1 个，受随身物 / 功德影响
```

保底：

```text
如果没有任何武器线索，系统授予“无名飞剑·残”。
如果生命低于阈值，系统授予“回春丹 x1”。
```

---

## 6. 敌人池

第一战敌人不要太多，但要有差异。

```text
域外残兵：基础小怪，直线移动。
裂隙飞虫：小型快速敌人，教横移。
虚空射手：远程发弹，教读弹。
业火残魂：若业力高，出现概率提高。
心魔幻影：若心魔/魔心暗种，特殊出现。
战场巡游者：小 Boss。
```

---

## 7. 小 Boss：战场巡游者

定位：第一场战斗高潮。

机制：

```text
Phase 1：慢速扇形弹
Phase 2：锁定雷线 / 虚空裂缝
Phase 3：残兵召唤 + 中央压迫
```

HP 目标：

```text
普通开局：45–75 秒击败
强开局：30–45 秒
弱开局：75–100 秒，但不应过分折磨
```

---

## 8. 天命展示事件

第一战必须至少触发一次“人生结果反馈”。

示例：

| 条件 | 事件 |
|---|---|
| 天妒英才 | 第一次顿悟选项品质提高，出现天道注视预警 |
| 苟道至尊 | 前 60 秒不放法术，生成潜修护盾 |
| 废灵逆命 | 首次濒死触发逆命回响 |
| 丹道奇才 | 初始回春丹强化，丹药炼化特效明显 |
| 古雷真血半觉醒 | 雷系击杀后小链雷 |
| 祖传玉佩 | 首次受击触发玉佩护盾 |

---

## 9. 掉落与奖励

第一战掉落要教学而不是刷资源。

```text
小灵气球
真元球
少量战场残晶
小 Boss 掉落系统家园核心
根据随身物掉落对应线索
```

战后结算：

```text
战场残晶
下品灵石
初始洞府材料
第一战称号 / 心结 / 伤痕
```

---

## 10. 失败处理

第一战失败不重置人生。

失败后：

```text
进入战败结算
添加“初战惊魂”心结，或若已有废灵逆命则转为逆命点
允许再次进入域外战场
给一个系统提示
若失败两次，可低保回春丹 / 护命符
```

失败文案：

```text
【初次征召失败。】
【人生模拟结果已保存。】
【宿主可再次接受域外战场投放。】
```

---

## 11. 战后进入家园

成功后：

```text
战场巡游者崩解
系统提取残余灵机
解锁系统家园权限
进入家园开辟界面
```

不要直接跳到完整洞府。要先有“开辟”仪式。

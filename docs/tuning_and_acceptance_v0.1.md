# 调参、验收与测试标准 v0.1

## 1. 体验验收

玩家在创建角色界面至少应产生三种明显感受：

```text
1. 这个词条很强，但代价很大。
2. 这个组合看起来有隐藏共鸣。
3. 我想再 Roll 一次。
```

## 2. 分布验收

用固定随机种子生成 10,000 次，检查：

| 指标 | 合格范围 |
|---|---:|
| 主天命奇命及以上 | 55%–70% |
| 主天命地命及以上 | 8%–15% |
| 主天命天命/逆命/禁命 | 0.5%–6% |
| 副天命凡命出现率 | 15%–35% |
| 劫命大劫/死劫 | 15%–25% |
| 硬互斥结果 | 0 |

## 3. 上头但不破坏平衡

高品质开局不应是无脑最优。检查：

```text
天命/逆命/禁命必须至少有 2 条明显负面。
地命以上应带有模式风险、寿元风险、心魔风险、雷劫风险、资源风险之一。
劫命不能被正面天命完全抵消，只能被缓和。
```

## 4. UI 验收

命格卡必须显示：

```text
品质
名称
描述
正面效果
负面效果
影响阶段
标签
互斥/共鸣/警告
```

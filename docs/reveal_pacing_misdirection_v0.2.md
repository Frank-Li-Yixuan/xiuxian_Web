# Reveal Pacing & Misdirection v0.2

## 1. 揭示节奏

隐藏内容不应一次性揭示，而应按照：

```text
模糊预兆
  → 多次同类异常
  → 与随身物/天命共鸣
  → 半年选择出现命选项
  → 玩法插曲验证
  → 半揭示
  → 18 岁解析
```

## 2. 误导机制

误导不是欺骗玩家，而是制造“可解释歧义”。

例如：

```text
雷雨骨痛
```

可能来自：

```text
古雷真血
劫雷亲和
雷厄缠身
天妒英才
系统战场回响
```

真正的隐藏命由内部 state 决定，但外部文案可以保持歧义。

## 3. 误导分辨方式

玩家通过长期事件慢慢分辨：

```text
雷雨 + 骨痛 + 雷法亲和 → 古雷真血
雷雨 + 病弱 + 寿元低 → 天妒英才 / 雷厄
雷雨 + 耳边系统杂音 → 系统前兆
雷雨 + 梦见战场 → 域外战场回响
```

## 4. 揭示规则

| 条件 | 效果 |
|---|---|
| hidden progress >= 60 | 允许半揭示 |
| progress >= 80 + 对应玩法插曲成功 | 允许强半揭示 |
| progress >= 100 | 允许真名揭示 |
| 18 岁系统解析 | 允许揭示、封印或不稳定觉醒 |
| 低心性 + 魔印/心魔类 | 可能不稳定揭示 |
| 高心性 + 清净类 | 可能稳定封印或净化 |

## 5. UI 显示策略

早期：

```text
你偶尔听见雷声深处有战鼓。
```

中期：

```text
你逐渐意识到，雷声似乎不是从天上传来。
```

半揭示：

```text
你的血脉似与雷有关，但这股力量并不稳定。
```

18 岁揭示：

```text
系统解析完成：古雷真血·未全醒。
```

# 《隐藏血脉、身世与随身物系统 v0.1》

版本：v0.1
定位：角色创建阶段的第三层随机系统，承接「开局属性与灵根随机生成器」和「天命词条系统」，为 0–18 岁人生模拟、18 岁系统觉醒、域外战场第一战和洞府开局提供叙事与机制钩子。

---

## 0. 系统目标

本系统解决三个问题：

```text
1. 玩家表面上是什么出身？
2. 玩家身上暗藏什么血脉、前世、封印或系统异常？
3. 玩家从凡俗人生中带着什么随身物进入 18 岁觉醒？
```

它的核心体验是：

> **表面普通，暗藏玄机。**
> 玩家创建角色时不一定知道自己抽到了什么大货，但能通过预兆、随身物、人生事件逐步推断。

这能满足修仙题材里非常重要的爽点：

```text
废柴其实是远古大能转世。
山村孤儿体内藏着古雷真血。
破落修士之后手里的木剑其实有前世剑魄。
药铺学徒梦见的丹炉，对应丹圣遗骨。
```

## 1. 与前置系统的关系

### 1.1 输入

本系统读取两个前置系统的结果：

```text
OpeningInnateDraft
  - 命盘类型
  - 六维先天资质
  - 精气神种子
  - 灵根类型
  - 五行/异灵根标签
  - destinyBiasTags / lifeEventBiasTags / hiddenFateBiasTags

DestinySelectionState
  - 主天命
  - 副天命 x2
  - 劫命/缺陷
  - 共鸣/相冲标签
  - 高风险/苟道/丹道/剑道等路线倾向
```

### 1.2 输出

本系统输出：

```text
OriginFateDraft
  - backgroundOrigin：表面身世
  - hiddenFate：隐藏血脉/前世/封印/系统共鸣，内部真实值
  - visibleHiddenOmen：玩家可见的隐藏预兆
  - carriedItems：随身物 1–2 个
  - lifeEventBiasTags：人生模拟事件偏置
  - age18ConversionHooks：18 岁觉醒转化钩子
  - modeProjectionTags：域外战场和后续模式投射标签
```

## 2. 三层信息：可见、半隐藏、真实

### 2.1 可见信息

创建角色页可以直接显示：

```text
出身名称
出身描述
随身物名称与描述
隐藏预兆文本
模糊风险提示
隐藏异动等级：隐约 / 异动 / 将醒
```

例如：

```text
出身：山村孤儿
你在青云山脚的偏僻村落长大，父母早亡，靠乡邻接济成人。

隐藏预兆：你偶尔能听见雷云深处的战鼓。
异动等级：命中异动
随身物：残破木剑、裂纹铜钱
```

### 2.2 半隐藏信息

通过天机推演、人生模拟事件、特定选择可以逐步显示：

```text
隐藏命类别：血脉 / 前世 / 封印 / 系统共鸣
相关标签：雷、魂、丹、剑、功德、魔印
风险方向：雷劫、心魔、短寿、宿敌
```

### 2.3 真实信息

创建页默认不显示：

```text
古雷真血
太阴残脉
龙骨未醒
丹圣遗骨
前世剑魄
系统共鸣体
魔印微痕
```

真实名通常在以下时机揭示：

```text
人生模拟中触发关键事件
18 岁系统觉醒结算
域外战场中濒死/顿悟/击杀精英
洞府开启后的特殊建筑/梦境事件
```

## 3. 表面身世系统

表面身世是玩家的凡俗开局，不只是背景文本。它影响：

```text
初始属性偏向
人生事件池
隐藏血脉权重
随身物池
家境/亲缘/师缘
开局资源
18 岁转化方向
```

### 3.1 第一批身世

| 身世 | 定位 | 机制倾向 |
|---|---|---|
| 山村孤儿 | 低资源、高韧性、隐藏大因果 | 生存、雷雨、前世、系统共鸣 |
| 药铺学徒 | 丹道、药理、火候 | 炼丹、药材、丹圣遗骨 |
| 道观杂役 | 经文、香火、道心 | 符箓、清心、系统/天书线索 |
| 猎户之子 | 肉身、山林、妖兽 | 炼体、兽缘、龙骨 |
| 破落修士之后 | 旧物、祖传因果 | 前世剑魄、天书、法宝线索 |
| 流民遗孤 | 战乱、业力、生存 | 心魔、魔印、功德选择 |
| 私塾童生 | 悟性、读书、经文 | 天书残页、法术研究 |
| 富户庶子 | 资源、人情、隐忍 | 气运、交易、系统共鸣 |
| 守墓人之子 | 阴气、神魂、生死 | 太阴残脉、魂修、心魔 |

### 3.2 身世不是强弱，而是路线

每个身世都应该有优势和劣势：

```text
富户庶子资源多，但家族冲突多。
山村孤儿资源少，但隐藏触发多。
药铺学徒适合丹道，但炼器弱。
守墓人之子魂修线索多，但阴冷/心魔事件更多。
```

## 4. 隐藏血脉 / 前世 / 系统共鸣

隐藏命分 5 类。

| 类别 | 说明 | 示例 |
|---|---|---|
| bloodline | 血脉/体质/骨相 | 古雷真血、太阴残脉、龙骨未醒 |
| pastLife | 前世残魂/前世记忆 | 丹圣遗骨、前世剑魄、域外战场回响 |
| curseSeal | 魔印/诅咒/封印 | 魔印微痕 |
| karmicSeed | 功德/因果种子 | 功德种子 |
| systemResonance | 系统异常适配 | 系统共鸣体 |
| karmicObject | 与随身物绑定的因果物 | 天书残页 |

## 5. 隐藏命进度

每个隐藏命有一个内部进度：

```text
hiddenFateProgress: 0–100
```

创建页只显示模糊等级：

| 进度 | UI 显示 | 含义 |
|---:|---|---|
| 0–29 | 隐约预兆 | 只有很淡的提示 |
| 30–69 | 命中异动 | 可明显感觉异常 |
| 70–99 | 封印将醒 | 人生模拟可触发半觉醒 |
| 100 | 已觉醒 | 18 岁或特殊事件可揭示 |

### 5.1 进度来源

进度来自：

```text
灵根匹配
出身匹配
天命匹配
随身物匹配
月度事件
半年重大选择
人生模拟中的成功/失败判定
```

例子：

```text
雷灵根 + 劫雷亲和 + 山村孤儿
→ 古雷真血权重提高，初始进度更高。

药铺学徒 + 丹道奇才 + 药铺铜炉
→ 丹圣遗骨权重和进度提高。

破落修士之后 + 残破木剑 + 天生剑骨
→ 前世剑魄权重和进度提高。
```

## 7. 生成流程

### 7.1 总流程

```text
输入 OpeningInnateDraft + DestinySelectionState
  ↓
根据权重抽表面身世
  ↓
根据身世/灵根/天命/属性抽隐藏命
  ↓
生成隐藏命初始进度
  ↓
生成随身物 1–2 个
  ↓
生成玩家可见隐藏预兆
  ↓
合成 lifeEventBiasTags / modeProjectionTags
  ↓
输出 OriginFateDraft
```

### 7.2 权重公式

身世权重：

```text
backgroundWeight =
  baseWeight
  + openingTagBonus
  + destinyTagBonus
  + spiritualRootBonus
  + lockBonus
```

隐藏命权重：

```text
hiddenFateWeight =
  baseWeight
  + sum(matchingBiasTags)
  - sum(matchingAntiBiasTags)
  + inspirationModifier
  + fortuneModifier
  + divinationPityModifier
```

随身物权重：

```text
itemWeight =
  baseWeight
  + originBias
  + hiddenFateSynergy
  + destinyBias
  + spiritualRootBias
```

## 8. UI 显示要求

创建角色页的「身世血脉」区域应显示：

```text
表面身世
身世描述
隐藏预兆 1–2 条
异动等级
随身物列表
模糊转化提示
```

不要显示：

```text
隐藏血脉真实名称
隐藏进度精确数字
完整战斗效果
完整洞府效果
```

推荐 UI 文案：

```text
【身世】药铺学徒
你自幼在药铺帮工，识得草木寒热，也见过许多凡人病苦。

【命中异动】
你闻到药香时，脑中会浮现陌生火候。

【随身物】药铺铜炉
这件旧物似乎能在十八岁觉醒时被系统重新解析。
```

## 9. 人生模拟中的作用

人生模拟阶段会读取：

```text
backgroundOrigin.lifeEventBiasTags
hiddenFate.lifeEventBiasTags
carriedItem.lifeEventTags
```

用于提高特定事件权重。

例如：

```text
药铺学徒 + 药铺铜炉 + 丹圣遗骨
→ 采药、分药、火候、炸炉、丹炉梦事件权重提高。

守墓人之子 + 黑骨短笛 + 太阴残脉
→ 夜梦、神魂、墓地、魂火、心魔事件权重提高。
```

## 10. 18 岁系统觉醒转化

18 岁时系统会扫描：

```text
出身
隐藏命进度
随身物
人生事件记录
功德/业力/心结/伤病
```

然后生成：

```text
OuterBattlefieldOpeningModifier
DongfuOpeningHook
LongTermKarmaTag
```

示例：

```text
检测到随身物：药铺铜炉
解析结果：残炉火候
域外战场：开局获得回春丹 x1
洞府开启：炼丹房初始火候经验提高
```

## 11. 测试验收

实现后必须满足：

```text
1. 同 seed + 同输入生成完全一致。
2. 创建页不显示隐藏命真实名称。
3. 随身物与出身/隐藏命有可解释关联。
4. 高灵感/高气运角色更容易获得高进度隐藏命，但不必然。
5. 隐藏命进度会影响 18 岁揭示概率。
6. 生成结果能输出 lifeEventBiasTags。
7. 生成结果能输出 age18ConversionHooks。
8. 数据校验脚本通过。
```

# 天命成立条件、互斥共鸣与变异命格 v0.1

## 1. 文档定位

本文件承接《九宫命盘、三才阴阳与属性因果 v0.1》。上一份文档解决了“九宫属性如何形成命理因果”，本文件解决：

```text
什么样的命盘能支撑什么天命？
如果抽中的天命与命盘不符，如何变异？
互斥、共鸣、相冲如何变成新的可玩命格？
天命如何在人生模拟中显化？
```

本系统不再允许天命纯随机平铺。

## 2. 核心目标

玩家在创建角色页看到的每一个天命，都应该让人觉得：

```text
这个命格是这个角色“命盘自然长出来的”。
```

而不是：

```text
系统随机给我塞了一张强卡。
```

## 3. 核心生成顺序

推荐流程：

```text
OpeningInnateDraft
  ↓
NinePalaceEvaluation
  ↓
DestinyCandidatePool
  ↓
EligibilityEvaluator
  ↓
ConflictSynergyEngine
  ↓
MutationResolver
  ↓
DestinySelectionState
  ↓
CharacterCreationViewModel
```

天命生成器必须读取九宫派生评分：

```text
talentScore
vesselScore
stabilityScore
destinyPressureScore
lateBloomScore
rebellionScore
```

## 4. 天命的四类条件

每个天命都有四类条件。

### 4.1 成立条件 eligibility

决定这个天命能否成立。

例如【天妒英才】：

```text
talentScore >= 85
或 悟性 >= 90
或 灵感 >= 90
或 悟性 + 灵感 >= 160
```

### 4.2 支撑条件 support

决定天命是否以原形出现，还是弱化/变异。

例如【天妒英才】需要：

```text
寿元低
或 天道压力高
或 心性不稳
```

因为它的核心不是“聪明”，而是“慧光过盛，命数难承”。

### 4.3 反相条件 anti

如果命盘与天命叙事核心相反，则不能直接出现原命格。

例如低悟性低灵感不能直接拥有【天妒英才】。

### 4.4 变异规则 mutation

不合理组合优先变异成更有故事的命格。

例如：

```text
低悟性低灵感 + 抽中天妒英才
→ 妄承天机
```

```text
高资质 + 抽中废灵逆命
→ 天骄遭厄
```

## 5. 为什么使用变异，而不是硬删除

硬删除会让玩家感觉：

```text
我 Roll 到了好东西，但系统不给我。
```

变异则会让玩家感觉：

```text
这个命盘出现了特殊故事。
```

这非常符合修仙爽点：

```text
废柴不一定真废。
天才不一定顺遂。
魔心也可能被清净镇住。
苟道也可能藏着杀机。
```

## 6. 互斥、软冲突、共鸣

### 6.1 硬互斥

硬互斥表示两个命格的叙事核心无法共存。

例如：

```text
天妒英才 × 大器晚成
苟道至尊 × 以战养战
```

### 6.2 可变异互斥

有些互斥不应删除，而应合成变异。

```text
魔心暗种 × 清净琉璃心
→ 净莲藏影
```

### 6.3 软冲突

软冲突保留，但显示风险提示。

```text
丹道奇才 × 器灵之眷
→ 两条局外资源路线分流
```

### 6.4 共鸣

共鸣是玩家最想 Roll 到的部分。

```text
天妒英才 + 劫雷亲和
→ 慧光引雷
```

```text
废灵逆命 + 百折不摧
→ 百折逆命
```

## 7. 创建页显示规则

天命卡牌需要显示：

```text
品质
名称
一句话描述
正面效果摘要
负面效果摘要
标签
共鸣提示
冲突提示
是否变异
```

但不要显示：

```text
具体内部权重
完整调试条件
隐藏血脉真实名称
```

调试信息只在 dev/debug 面板展示。

## 8. 与人生模拟的关系

每个核心天命至少有：

```text
童年显化
少年考验
青春关键事件
成年结算投射
```

天命不是静态卡牌，而是人生中的反复显影。

## 9. 与玩法插曲的关系

天命会影响半年选择触发玩法插曲的种类。

例如：

```text
劫雷亲和
→ 更容易触发雷雨后山 STG 插曲

丹道奇才
→ 更容易触发守药田/炼丹/虫潮割草插曲

魔心暗种
→ 更容易触发心魔 DBG / 禁忌选择

器灵之眷 + 天生剑骨
→ 更容易触发飞剑灵识 STG 插曲
```

## 10. 与 18 岁觉醒的关系

18 岁时，天命将被投射为：

```text
域外战场第一战修正
系统家园初始线索
局外修行长期标签
```

例如：

```text
天妒英才
→ 第一次顿悟品质 +1，但第一战高危事件 +1

苟道至尊
→ 生存奖励提高，主动法术伤害略降

废灵逆命
→ 首次濒死触发逆命回响
```

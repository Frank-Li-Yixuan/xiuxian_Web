# 变异命格设计目录 v0.1

## 1. 变异规则

变异发生于三种情况：

```text
1. 抽中的天命与命盘反相。
2. 两个互斥命格可融合为特殊命格。
3. 命格缺少支撑条件，需要弱化为合理版本。
```

## 2. 第一批变异

### 妄承天机

来源：天妒英才反相。

适用：悟性/灵感不足，却被天道压力或系统异常缠上。

玩法：偶有启示，但不稳定，灾兆多于收益。

### 慧极易折

来源：天妒英才支撑不足。

适用：确实早慧，但承载不足。

玩法：学习快，但过度学习会伤身。

### 天骄遭厄

来源：废灵逆命反相。

适用：高资质却抽到逆命。

玩法：失败来自外劫，破劫后恢复天资。

### 隐杀之命

来源：苟道至尊与杀伐倾向冲突。

玩法：越久不出手，第一次出手越强。

### 血惊未定

来源：以战养战缺少心性支撑。

玩法：危险可成长，但首次见血会形成心结。

### 雷厄缠身

来源：劫雷亲和承载不足。

玩法：雷雨与天劫更频繁，但收益不稳定。

### 药性亲和

来源：丹道奇才弱支撑。

玩法：不是丹圣之资，但适合药理。

### 器物缠魂

来源：器灵之眷与神魂/旧物风险相冲。

玩法：旧物回应你，但回应的可能是残魂。

### 静极生执

来源：坐忘常定与心结/执念倾向结合。

玩法：静修强，但会执着。

### 净莲藏影

来源：魔心暗种 × 清净琉璃心。

玩法：清净镇魔，魔影可控爆发。

### 孤星护命

来源：天煞孤星与功德/护生冲突。

玩法：孤星不害人，反成护命之星。

## 3. 变异 UI 呈现

不要告诉玩家：

```text
这是因为你抽到 X 但不合格，所以变异。
```

可以写：

```text
天机流转中，此命格出现异相。
```

玩家卡牌上只显示变异后的命格。

Debug 面板可显示来源。
